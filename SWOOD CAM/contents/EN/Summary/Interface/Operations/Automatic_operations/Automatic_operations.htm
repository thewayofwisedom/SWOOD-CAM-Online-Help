AUTOMATIC OPERATIONS

Introduction

Automatic operations are based on the geometric recognition of the model.In addition to automatically detecting geometries, they also allow customization of machining strategies when creating machining entities.

General

  1. Selection

The automatic operations all have the same creation methodology. They do not require any selection (body / surface / edge) to be created. In a part or assembly, it is therefore necessary to click  in the "empty"/background part of SOLIDWORKS before creating its operation.

 

  1. Navigation

When SWOODCAM has detected several profiles to be machined, it is possible in the automatic operation to navigate between the profiles using the following buttons :

  1.  Selecting previous profile
  2.  Selecting next profile

  Automatic drilling

The automatic drilling allows, with the help of the automatic geometry detection SWOOD CAM, to process cylindrical "holes", through holes or not.

Note that SWOODCAM will carry out the machining of all geometries that can be processed by the tool used.

  Drilling

After choosing the tool and the drilling operation, the operation setting window will appear on the screen:

The principle remains the same as with the manual drilling. However, with the automatic operation, are managed the blind or through-all cases simultaneously.

Options related to the operation :

Depending on how the operation is set up, certain parameters can be entered:

  1.  Geometries depth

    Allows the hole to be drilled from the top side of the hole to the bottom side..

    1. Approach Z value
    2. Bottom oversize (blind case)
    3. Bottom exceeding (through-all case)
  2.  Extension to the raw :

    Allows the hole to be drilled from the top side of the raw to the bottom side.

    1. Approach Z value
    2. Bottom oversize (blind case)
    3. Bottom exceeding (through-all case)
  3.  Defined depth :

    Allows you to drill from the top of the holes to the selected depth :

    1. Approach Z value
    2. Bottom oversize
    3. Defined depth
  4.  Extended to rough top and bottom :

    Allows the holes to be drilled using the top side of the raw to the bottom side of the raw.

    1. Approach Z value
    2. Bottom oversize (blind case)
    3. Bottom exceeding (through-all case)

The graphic area :

The graphic area is empty if the tool used is a simple tool (because tool choice made before choosing the operation).

In case a drilling block is used, the graphic area will display its schematics of it.

  1. The spindles used are displayed in blue. 
  2. Unused spindles are grayed out. 

It is not possible to choose which tool to use by clicking on it from the graphic area. As previously mentioned, the automatic operation processes all possible geometries.

Tips : If a spindle is not to be used, one solution is to remove it from the drill block, or replace it with an empty spindle.

To the right of the graphic area there is an additional option :

  1.  Reverse boring direction : Reverse the drilling direction. The spindle orientation will also be reversed.

  Machine pocket milling

The Machine Pocket milling is an operation for processing the blind geometries (without islands) present in a part. Contrary to the Pocket Milling, this operation does not export a trajectory but allows you to export a recess macro with the parameters indicated in SWOOD CAM.

This operation is available for export to the following software:

  • BiesseWorks
  • BSolid
  • IMAWop
  • Maestro
  • TPA

  1.  Approach Z value
  2.  Maximum depth of passes
  3.  Depth of the last passes
  4.  Bottom oversize (positive or negative values accepted)
  5.  Extra thickness lateral (positive or negative values accepted)

Extension to the raw : Automatically detects the height of the profile to be processed up to the raw part.

  1.  Extension to the raw deactivated
  2.  Extension to the raw activated

  WOODWOP Pocket milling

WOODWOP Pocket milling is an operation to process blind geometries in a part. Contrary to the operation Pocket Milling, this operation does not export a trajectory but allows to export in WOODWOP a pocket macro with the parameters indicated in SWOOD CAM.
This operation is available for any free form, except for pockets with islands.

The data to be entered to configure the operation is exactly the same as that of the WOODWOP Pocket milling.

Parametrization of the geometries to be processed:

You can filter the detection parameters of the operation by "Edit Selection" from Machining Pocket milling

 

  Auto Clamex

The Automatic Clamex enables the automatic processing of clamex geometries on the part / assembly.

The data to be filled in are exactly the same as those of the Clamex Milling. 

After creating the automatic clamex operation, it is possible to filter the geometries to be processed via the machining entity of the automatic clamex.

  Auto flat face

Automatic Sawing allows for the automatic processing of flat, inclined or non-inclined surfaces in the part/assembly.

The only operation available is sawing  . The data to be filled in are exactly the same as those of the Saw milling.

After creating the Auto flat face operation, it is possible to filter the geometries to be processed via the machining entity of the Auto flat face.

  Auto groove

The Automatic Slot allows you to automatically process grooves in the selected plane, whether they are blind or not.

  Cut-groove

  Offcut

The automatic machining entity "Offcut" allows to detect the reusable offcuts generated during the Nesting process and set in the Boards Library.

Available operations

The following operations are available (depending on the tool selected):

  •  Move tool center.
  •  Contour.
  •  Partial contouring.
  •  Saw.

 

Specific parameters

When using the "Offcut" machining, the specific offcut sorting parameter is available:

  • Both offcuts: Will take into account the two offcuts in the operation.
  • First offcut: Will take into account only the first offcut in the operation.
  • Second offcut: Will only take into account the second offcut (if there is one) in the operation.
