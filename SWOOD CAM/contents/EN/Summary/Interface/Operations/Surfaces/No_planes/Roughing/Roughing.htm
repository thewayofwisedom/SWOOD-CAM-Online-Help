OPERATION: ROUGHING

WHAT IS THE ROUGHING OPERATION?

The " Roughing" operation is an operation allowing to roughen complex shapes. It is usually followed by another operation of the machining " Volume", the operation " Finishing". This operation only works in tool center, without radius compensation mode.

 

SELECTION

The "Roughing" operation is linked to the SolidWorks  Solid Body. So there are two different selection modes depending on the number of bodies the part has:

  • A single Solid Body: No selection is necessary: in the SolidWorks menu bar, click on : Tools / SWOOD CAM /  3D Milling.
  • Several Solid Bodies: One body must be selected: in the FeatureManager design tree, expand the  Solid Bodies folder and select one, then in the SolidWorks menu bar, click on : Tools / SWOOD CAM /  3D Milling.

 

SPECIFIC PARAMETERS

 Each time a parameter of the "Roughing" operation is modified, it will be necessary to  rebuild the operation in order for the modification to be taken into account. If one of the parameters has been modified, but the operation has not been rebuilt, there will be a warning sign on the  rebuild button and on the " Roughing" operation.

 

Approach parameters

In the "Roughing" operation there is no approach to choose.

 

Tool pass parameters

The tool pass parameters specific to the "Roughing" operation are path parameters:

  • Tool thickness %: Overlap in percentage of tool diameter between two passes.
  • Path lead in mm: Overlap in millimeters between two tool passes.

 

Dimensional parameters

The dimensional parameters specific to the "Roughing" operation are as follows:

  • Zmax in mm: Detected by default, this is the highest point of the volume in relation to the program origin.
  • Surface extension mm: Allows to create a lateral overlap of the tool around the volume body (not activated by default).
  • Approach Z value: Tool clearance height value before making fast movements.
  • Vertical offset: Corresponds to the quantity of material to be left vertically, which will have to be reworked by a finishing operation (in mm).
  • Horizontal offset: Corresponds to the quantity of material to be left horizontally, which will have to be reworked by a finishing operation (in mm).
  • Maximum depth of passes: Maximum height that the tool can machine in one pass. Used to calculate the number of passes.
  • Zmin en mm : Detected by default, this is the lowest point of the volume in relation to the program origin.

 

Overlap strategy

  •  Enables additional contouring during the roughing operation.
  •  Disables additional contouring during the roughing operation.

 

Precision

Maximum amount of material remaining after roughing (without extra thickness).

  •  Minimum precision.
  •  Intermediate precision.
  •  Maximum precision.

 

Milling area

Permet de choisir quel élément définira la zone d’ébauche.

  •  Stock (by default): Takes the smallest rectangular parallelepiped enclosing the volume of the workpiece.
  • Body: Allows to select one of the existing Solid Bodies in the part.
  • Sketch: Allows to select one of the existing sketches in the part.
