FINISHING

 

Introduction

The Finishing operation is an operation allowing complex surface shapes. It generally follows a roughing operation. This feature works only in the tool centre, without compensation mode.

This operation requires the use of a spherical type tool.

 

Selection

In order to limit the surfaces to be machined, it is advisable to machine surface bodies created in SWOLIDWORKS, such as knit surfaces.

  • Single volume body : Without making a selection, select   3D Milling from the SOLIDWORKS Tools/SWOOD CAM menu.
  • Surface body : Select the surface body to be machined in the SOLIDWORKS Feature Manager and then select  3D Milling from the SOLIDWORKS Tools/SWOOD CAM menu.

 

Once the tool and the Finishing milling operation have been selected, the operation configuration window will appear on the screen. 

For details on the data common to all operations, see the SWOODCAM Millings operations page.

 

 

Machining parameters

Unlike standard SWOOD CAM operations, this operation requires each parameter to be configured independently.

By selecting this option, the following diagram will appear on the screen, allowing you to customise the various surface finishing parameters. 
 

Machining parameters

  1. Surface extension in mm: Lateral extension of the tool around the surface body.
  2. Path lead in mm: Path intervail made by the tool (in mm). Changing this parameter causes the Peak Height parameter to be recalculated.
  3. Peak height: Size of the peak that will be left by the tool after machining. This value depends on the Path lead as well as the tool diameter. Changing this parameter will recalculate the Path lead parameter.
  4. Approach Z value: The point at which the tool will be at milling speed. This distance is added to the machining depth starting from the selected edge.
  5. Extra thickness lateral: Corresponds to the quantity of material to be left (in mm).

   Inversion of the profile: Reverses the surface normal.

 

Distribution uniform/not uniform

This feature is used to improve the surface finishing. Instead of having an uniform distribution (default) it distributes the paths with the same space between them.

  •  Distribution uniform (Default value)
  •  Distribution not uniform (Optimized)

           A positive value may be entered in the textbox "minimal space between step”

           

           The smaller the value, the better the result.

 

3-axis/5-axis machining : The finishing path varies depending on which mode is used (3-axis / 5-axis). In the case of 5-axis machining, the tip of the ball  can remain in constant contact with the surface, which is not possible in 3-axis machining.

  1.  3-axis machining : Enabled by default, the path is calculated to leave the edge of the hemispherical milling cutter in contact with the surface to be machined.
  2.  5-axis machining : The trajectory is calculated so that the tip of the hemispherical milling cutter is in contact with the surface to be machined. Only possible with a 5-axis head.
    Enabling this option will display the parameter Angle B Max, to block the angle that the machining head can take.

 

 

Path direction

4 types of surfacing are available:

  •  X Direction : The default setting is to use the direction of the X-axis.
  •  Y Direction : The trajectories are in the direction of the Y-axis.
  •  Any Direction : Use to specify a path angle with respect to the X-axis.
  •  Path by level : The trajectory is not in a specific direction, but SWOOD calculates a depth-dependent trajectory. Machining is carried out from the highest point to the lowest point. This type of machining is recommended for concave or convex shapes (half-spheres, sinks...).

 

Path orientation

  •  Zig-zag : Default setting, the tool moves in and out in the selected direction.
  •  Zig-zag path with jump : The tool goes in and out in the chosen direction, while adding up and repositioning the tool before each change of direction.
  •  One way : Forces the tool to always follow the same machining direction.
