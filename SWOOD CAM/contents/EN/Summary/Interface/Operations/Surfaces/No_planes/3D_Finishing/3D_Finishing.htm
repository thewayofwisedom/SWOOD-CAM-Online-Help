OPERATION: 3D FINISHING

WHAT IS 3D FINISHING?

The 3D Finishing operation is used to machine complex surfaces. It generally follows a roughing operation. This operation only works in tool center, without compensation mode.

This operation, directly linked to its machining, is managed entirely in the editing window, with the result of modifications displayed directly in its graphic view.

 

ACCESS 3D FINISHING

To create a  3D Finishing operation, you must :

  • Open a SolidWorks part.
  • Go to the SolidWorks menu bar: Tools → SWOOD CAM →  3D Finishing.

 

3D FINISHING INTERFACE

The new 3D finishing window is divided into 5 sections:

 

Toolbar

The toolbar of the new 3D finishing window contains the following commands:

Icon

Command name

Description

Technology

Defines the machining and technology parameters of the operation. These include the operation name as it will appear in the milling phase, tool selection and cutting parameters. It is also possible to choose the operation's technology parameters and manage the Z axis movements.

Geometry

Selects the faces to be machined and the faces to protect. By default, the entire 3D model is selected. It is also possible to select only some of the model's faces. Define also surfaces to be protected by selecting 3D model faces and by safety offset.

Constraints

Defines operation reference planes and their offsets, such as Z security, Z approach, maximum and minimum machining altitude planes.

Uses also sketches to define specific areas for machining, inclusive or exclusive.

Passes

Defines the operation's pass parameters, such as passes type or direction. It also defines the management of steep areas of the 3D model and the tool tilting. In this section, you can define the passes stepover.
Transitions Defines transition parameters between passes.
Rebuild Recalculates operation according to last parameter modification.

When a command is clicked, the PropertyManager adapts to the command and displays the section parameters.

 

Validation bar

The validation bar contains the following commands:

Icon

Command name

Description

Operation Validation

Validates parameters and creates a machining operation.

Operation Cancel

Does not take machining parameter modifications into account. If this is the first edition of the operation, it is not created; if it is an edition of an existing operation, the parameter modification will not be taken into account.

Previous Section

Returns to previous section.

Next Section

Skip to next section.

The reconstruction bar is a visual aid to help you see how far along you are in the reconstruction of the operation, since modifying parameters alone is not enough to update the operation.

A green reconstruction bar means that the model is correctly reconstructed, according to the latest parameter modifications.
A blue reconstruction bar means that the model has not yet been reconstructed.

 

PropertyManager

The parameters displayed in the PropertyManager depend on the section selected in the toolbar.

 

Status bar

The status bar displays messages tailored to the status of the model and its reconstruction.

 

Graphics Area

The graphics area allows you to display and interact with the 3D model, in particular when selecting surfaces to be machined, protected or excluded.
The 3D model is displayed without texture, with its origins defined in the milling phase.

3D FINISHING PARAMETERS

The parameters for each parameter section are as follows:

 Technology

Parameter name

Parameter description

Operation Name

Renames the machining operation as it will appear in the SWOOD CAM milling phase.

Tool

Displays the name of the selected tool. Two selection modes are available: If you click on the selection box, you have access to compatible tools already present in the milling phase; clicking on the cursor opens a small selection window, giving access to all tools in the library.
Linked to tool cutting parameters

Uses parameters as defined in the tool library.
Tool Frequency Tool rotation frequency in rotations per minute.
Tool Feed Rate Tool feed rate in meters per minute.
Diving Feed Rate Diving speed of the tool in the material in meters per minute.
Suction Cap Position Defines the position of the suction hood during machining. Several positions are available: The "None" position does not activate the suction hood at all; the "High" position places the suction hood in its highest position; the "Middle" position places the hood between the highest and lowest positions; the "Low" position moves the suction hood to its lowest position; the "Custom" position places the suction hood in a specific position, defined by a number (depending on the machine used).
Spoiler Activates or deactivates the spoiler when the tool is in use.
Z Approach Mode Allow to configure fast machine movements between two profiles.
Z Security Clearance When Finished Enables additional end-of-operation clearance safety behavior. Linked to "Safety Z" parameter.
Z Security Base Plan Defines the base plane, from which the Z security is defined.
Z Security Offset Defines the offset of the base plane, which will define the safety Z.
Z Security Displays the final height of the safety Z in relation to origin OP0.
Z Approach Base Plan Defines the base plane, from which the Z approach is defined.
Z Approach Offset Defines the offset of the base plane, which will define the Z approach.

 

 Geometry

Parameter name

Parameter description

Full Body

Selects all faces of the 3D model.

Selected faces

Selects the face(s) to be machined.
Include Boundaries

Includes the boundaries of selected faces in the machining process.
Surface Offset Defines the offset of the selected surface from the 3D model.
Offset Defines the machining offset from protected surfaces.

 

 Constraints

Parameter name

Parameter description

Z Min Base Plan

Defines the base plane, from which the start of machining is defined (Z Min).

Z Min Offset

Defines the offset of the base plane, which will define Z min.
Z Max Base Plan

Defines the base plane, from which the Z max is defined. Z Max allows to select whether to stop machining before the end of the part height.
Z Max Offset Defines the offset of the base plane, which will define Z max.

 

 Passes

Parameter name

Parameter description

Precision

Defines the precision of the grid that the algorithm will use to create tool paths. Several options are available: "Low", gives a low-density grid; "Medium", gives a medium-density grid; "High", gives a high-density grid; "Auto", gives a grid whose density is based on the tool radius.

Passes Type

Defines the pass type, between parallel passes (in vertical planes) and contour passes (in horizontal planes).
Passes Direction

Defines the tool path angle relative to the axis direction.
Passes Scheduling Defines the machining order of the paths generated by slicing. Several options are available: "Cutting order", follows exactly the order in which the model was sliced; "Reverse cutting order", follows the reverse order in which the model was sliced; "Alternate cutting order", follows the slicing order, alternating the start of one slice with the end of the next; "Nearest cuts", searches for the nearest slice to the one just processed; "Closest paths", finds the path closest to the one just created.
Stepover Value Defines the distance between two passes.
Scallop Height Defines the height of the crests that will be left as a result of the tool sweep. This value is linked to the slicing step and the tool diameter percentage.
Stepover Tool Diameter Percent Defines the percentage of tool overlap between two passes.
Passes Density Allows to adjust the passes according to the curvature of the 3D model.
Minimum Stepover Defines the minimum distance between two passes (modifiable only if custom passes density is selected).
Maximum Stepover Defines the maximum distance between two passes (modifiable only if custom passes density is selected).
Tool Tilting Defines the positioning of the tool according to the trajectory. Two options are available: Fixed (default value), machines the part with the tool always vertical. Variable (5 axis) (only possible with a 5-axis machine), machines the part with the tool positioned perpendicular to the path.
Angle Limit (Can only be modified when the tool direction is "Variable (5 axis)") Defines the maximum angle B for blocking the angle the machining head can take.
Start Offset Value Defines the offset distance before the passes area.
Start Offset Percent Defines the percentage of offset before the passes area.
End Offset Value Defines the offset distance after the passes area.
End Offset Percent Defines the percentage of offset after the passes area.

 

  Transitions

Parameter name

Parameter description

Transition Type

Defines the transition mode between two trajectories. Several transitions are possible: "Up" transition produces an upward transition in the Z direction, where the altitude between entry and exit is equal; "Ramp" transition produces an upward transition in the Z direction, where there is a ramp between entry and exit; " Constrained " transition produces a transition that bypasses volume constraints instead of flying over them; " Straight " transition produces a straight transition between input and output; "Up to Z security" transition produces a transition up to the height of the Z security.

Transition Clearance

(Applicable only when the tool tilting type is "Variable 5 axis") Moves the tool back, keeping its axis, before making the transition.
Transition Offset

Adds a vertical offset before transition.
Constraints Transition Type (Applies only to sketch constraints) Defines the transition mode between two trajectories. Several transitions are possible: "Up" transition produces an upward transition in the Z direction, where the altitude between entry and exit is equal; "Ramp" transition produces an upward transition in the Z direction, where there is a ramp between entry and exit; "Constrained" transition produces a transition that bypasses sketch constraints instead of flying over them; "Straight" transition produces a straight transition between input and output; "Up to Z security" transition produces a transition up to the height of the Z security.
Constraints Transition Clearance (Applicable only for sketch constraint and when tool tilting type is "Variable 5 axis ") Retracts the tool, keeping its axis, before making the transition between two trajectories.
Constraints Transition Offset (Applies only to volume constraints) Adds a vertical offset before transition between two trajectories.
