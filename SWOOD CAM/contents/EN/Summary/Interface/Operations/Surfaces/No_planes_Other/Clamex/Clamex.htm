CLAMEX

 

Introduction

The Clamex operation is an operation to machine Lamello brand "Clamex" with a disc-type cutter. This operation can be created manually or automatically.

 

Selection

By selecting the content of a clamex geometry, it is possible to machine it by clicking  on the Clamex icon  and Clamex icon after selecting the blade, or via the menu Tools / SWOOD CAM / Clamex.

Once the tool and the Clamex milling operation have been selected, the operation configuration window will appear on the screen. 

For details on the data common to all operations, see the SWOODCAM Millings operation page.

 

 

Machining parameters

By selecting this option, the following diagram will appear on the screen, allowing you to customise the path of the contour, using the top and side faces of the contour as a reference.

  1. Approche Z value
  2.  Entry: Lateral penetration of the tool
  3.  Extra thickness total (total dimension of the clamex)

    4.  Approach distance

 

  1.  Rotation of the C axis by 180° : During a Clamex operation, the head is always positioned outside the part by default to avoid collision problems. The direction can be reversed if necessary.
