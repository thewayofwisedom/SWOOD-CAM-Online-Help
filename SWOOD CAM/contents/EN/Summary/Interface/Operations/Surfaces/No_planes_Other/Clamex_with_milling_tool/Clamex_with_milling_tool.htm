OPERATION: CLAMEX WITH MILLING TOOL

WHAT IS CLAMEX WITH MILLING TOOL?

The  Clamex with milling tool operation is used to machine grooves where Clamexes will be used as connectors, when the machine does not have an angular gear unit or when the groove is not accessible by the latter.


Example of Clamex machining with a milling tool

 

OPERATION APPLICATION

The  Clamex with milling tool operation can be applied to a manual or automatic  Clamex machining.

 

Operation creation

Simply select the "Clamex with milling tool" operation and "add" it.

 

OPERATION PARAMETERS

The  Clamex with milling tool operation contains the following specific parameters:

 

Lead In & Out distance

The lead in & out distance corresponds to the arc length of the machining outside the Clamex.

 

3 / 5 axis milling

The machining button is used to switch between 3-axis and 5-axis machining.

3 axis milling

5 axis milling

 

 
Angles B and C remain at zero during machining.

Angles B and C change to keep the tool axis perpendicular to the surface to be machined.
