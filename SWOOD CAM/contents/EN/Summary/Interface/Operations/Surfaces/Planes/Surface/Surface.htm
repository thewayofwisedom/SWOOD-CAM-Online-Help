SURFACE

 

Introduction

Surface is an operation that allows a flat surface to be treated to create a constant Z trajectory over the entire surface. It can be used, for example, to surface CNC spoilboard.

 

Selection

  1. By selecting the bottom of a pocket, it is possible to machine a pocket by clicking  on Milling .
  2. By selecting a sketch  in the SOLIDWORKS Feature manager tree and clicking  on Milling .
  3. By selecting a closed outline from the SolidWorks graphical interface and clicking  on Milling .
  4. By selecting a closed loop (edges) from the SolidWorks GUI and clicking  on Milling .

Once the tool and the Pocket Milling have been selected, the operation configuration window will appear on the screen.

For details on data common to all operations, see the SWOODCAM Milling page.

 

 

Machining parameters

Profile settings

By selecting this option, the following diagram will appear on the screen, allowing you to customize the machining parameters related to the profile.

 

  1.  Approach Z value
  2.  Maximum depth of passes
  3.  Depth of the last passes
  4.  Bottom extra thickness
  5.  Profile height 

As this operation is executed on a flat surface, the excess thickness to be machined is zero. This must be indicated if one of the parameters such as Maximum depth of pass, depth of last pass, or bottom oversize are used

Types of path

  •  Controlled bleed:
    As the default setting, the side overflow is set by the user.
  •  Centered path: The software establishes a trajectory centred on the width of the surface. The lateral overhang is not changeable and corresponds to a tool diameter.
  •  Bleed centered controlled: The software establishes a trajectory centred on the width of the surface. The side overflow is set by the user.

 

Path orientation

  •  Unique:
    Default setting, the tool rises with each pass to create a path in a single direction with respect to the X axis.
    1. Approach distance, outside the surface
    2. Latéral overflow in mm
    3. Angle of dip in relation to the X-axis
    4. Path lead in mm
    5. Path lead in % of diameter tool
  •  Zig-zag:
    This parameter makes it possible to have a reverse path, without tool retraction.
     
    1. Approach distance, outside the surface
    2. Latéral overflow in mm
    3. Angle of dip in relation to the X-axis
    4. Path lead in mm
    5. Path lead in % of diameter tool
  •  Spiral:
    This third parameter creates a concentric path from the outside to the inside of the panel.
     
    1. Approach distance, outside the surface
    2. Latéral overflow in mm
    3. Path lead in mm
    4. Path lead in % of diameter tool
