OPERATION: WASTE CUTTING

WHAT IS WASTE CUTTING?

The  Waste cutting operation is a tool center operation from an automatic machining entity, which is used to cut the waste between the parts on a Nesting plate into small pieces.

 

ACCESS TO WASTE CUTTING

To access the " Waste cutting " operation, you must not make any selection, click on the " Outline" button. Then, after selecting a tool, select the " Waste cutting" operation in the list of operations.

 

SPECIFIC PARAMETERS

Approach parameters

The approach parameters for the "Waste Cutting" operation are as follows:

  •  Axial approach: Approach where the tool plunges into the material from the top of the part.
  •  Approach ramp: Approach where the tool plunges progressively into the material from the top of the part at an angle.

 

Tool pass parameters

The specific tool pass parameters for the "Waste Cutting" operation are as follows:

  • Extra thickness total: Total height to be machined.
  • Approach Z value: Tool clearance height value before making fast movements.
  • Maximum depth of passes: Maximum height that the tool can machine in one pass. Used to calculate the number of passes.
  • Bottom exceeding: Depth in addition to the total height to ensure clean machining of the underside.

 

Dimensional parameters

The specific dimensional parameters of the "Waste Cutting" operation are as follows:

  • Minimum distance between two horizontal lines: Minimum spacing between two horizontal lines of the operation.
  • Minimum distance between two vertical lines: Minimum spacing between two vertical lines of the operation.
  • Minimum line size: Length below which a line will not be drawn. Avoids milling of too small lines.
  • Offset: Distance between the edge of the plate or a part, and the edge of a line of operation.
  • Length of the approach ramp (available only with approach ramp): Horizontal distance from the approach ramp.
  • Length of the release ramp (available only with approach ramp): Horizontal distance from the release ramp.

 

Order of passes

The specific parameters for the order of passes in the "Waste Cutting" operation are as follows:

  •  Machining by Profile: All passes of a profile are machined in sequence, before moving on to the next profile.
  •  Machining by Level: Each pass level is machined for all profiles, before moving on to the next level.

 

Profile sorting

It is possible to sort out which profiles will be taken into account in the operation:

  •  Exterior and interior.
  •  Exterior only.

 

Extension to the raw

The extension to the raw parameters specific to the "Waste Cutting" operation are as follows:

  •  Upper part, does not take into account the rough and the "Profile Height" is modifiable.
  •  Upper part, takes into account the raw and the "Profile Height" is not modifiable.
