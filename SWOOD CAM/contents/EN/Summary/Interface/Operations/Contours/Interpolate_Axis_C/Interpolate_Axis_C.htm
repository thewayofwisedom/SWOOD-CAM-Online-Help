INTERPOLATE AXIS C

 

Introduction

Interpolate Axis C milling allows a profile to be machined by fixing the B axis and allowing the C axis to be rotated. It is generally used with an aggregate. This machining is encoded in tool center, without compensation.

 

Selection

By selecting a face of a profile or edge, you can create an Interpolate Axis C milling by clicking  on milling . If a face is slected, a propagation window will appear on the screen, allowing you to manage the  manage the propagation of the selection :

  1.  Select previous face : Allows to propagate the selection to "the left side of the current selection".
  2.  Select next face : Allows to propagate the selection to "the right side of the current selection".
  3.  Select full profile : Allows to propagate the selection to all the profile.
  4.  Select bottom : Allows you to propagate the selection to the bottom face of the pocket, if it exists.
  5.  Cancel previous selection : Cancels the previous selection.
  6.  Validate the selection : Allows you to validate the propagation of the selection to which you want to apply the milling.
  7.  Cancel selection : Cancels the selection.

Once the tool and the Interpolate Axis C milling operation have been selected, the operation configuration window will appear on the screen. 

For details on the data common to all operations, see the SWOODCAM Millings operations page.

 

 Reference at the bottom of the profile :

  1. Extra thickness lateral (positive or negative values accepted)
  2. Offset Z (in relation to the bottom of the profile, positive or negative values accepted).
  3. Approach Z value
  4. Angle of the tool / vertical: Angle of the tool to the Z-axis (fixed value if the tool is an agregate with B fixed)

 

 Reference at the top of the profile :

  1. Extra thickness lateral (positive or negative values accepted)
  2. Offset Z (in relation to the bottom of the profile).
  3. Approach Z value
  4. Angle of the tool / vertical: Angle of the tool to the Z-axis (fixed value if the tool is an agregate with B fixed)

 

 Reverse direction : Reverses the direction of the tool/spindle.
