CHAMFER

 

Introduction

Chamfer milling allows a chamfer to be machined via an edge selection. This operation is generally used with an aggregate, and does not require a drawn chamfer. This machining is encoded in tool center, without compensation.

 

Selection

By selecting an edge or a sketch, you can use the Chamfer by clicking  on Milling  . Once you have selected the tool and the Chamfer milling, the operation configuration window will appear on the screen.

For details on the data common to all operations, see the SWOODCAM Millings operations page.
