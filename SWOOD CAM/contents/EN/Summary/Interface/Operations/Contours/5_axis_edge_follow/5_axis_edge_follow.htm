5-AXIS EDGE FOLLOW

 

Introduction

5-axis edge follow is an operation allowing to follow a non-planar profile or a profile requiring interpolation of the 5-axis head along an edge. It is available via edge selection. This feature works only in tool center mode, without compensation mode.

 

Selection

By selecting an edge, it is possible to perform 5-axis edge follow by clicking  on milling  and 5-axis edge follow after selecting a tool.

Once the tool and the 5-axis edge follow milling operation have been selected, the operation configuration window will appear on the screen. 

For details on the data common to all operations, see the SWOODCAM Millings operations page.

 

Machining parameters

Contrary to standard SWOOD CAM operations, this operation requires each parameter to be configured independently.

Profile settings

By selecting this option, the following diagram will appear on the screen, allowing you to customise the contour path, using the selected edge as a reference. 
Because machining is edge-related, there is no material detection, so machining depths and Z approach must be indicated to prevent rapid plunge into the material.

 

  1.  Approach Z value: The point at which the tool will be at milling speed. This distance is added to the machining depth starting from the selected edge.
  2.  Maximum depth of passes: Allows the profile to be machined in several passes. The Extra tchickness total must be entered.
  3.  Bottom exceeding (positive or negative values accepted): Tool extending distance from the selected edge.
  4.  Extra thickness lateral (positive or negative values accepted): Allows the machining to be shifted laterally.
  5.  Extra thickness total (to be specified manually): Indicates the height/depth of the machining  to be performed.

Appraoch and Disengagement parameters

As the machining is related to an edge, there is no material detection, so the appraoch and disengagement distances must be indicated in order to avoid any rapid movement into the material.

 

 

  1.  Appraoch distance: Approach distance of the tool, normal to the selected approach point.
  2.  Length of the appraoch ramp: Length of the approach tangent to the edge.
  3.  Length of the release ramp: Length of the disengagement tangent to the edge.
  4.  Disengagement distance: Disengagement distance of the tool, normal to the selected approach point.

Toolpaths and trajectories

In relation to an edge, several toolpaths can be defined according to different parameters of normals, tangents.

 

Tool path : A toolpath can be defined as 2 normal or 2 tangent toolpaths, detected in relation to the selected edge.

 Toolpath along Normal 1: Allows you to work with the tip of the tool, relative to the first normal detected (usually above the selected edge).

 

 Toolpath along Tangent 1: Allows you to work with the edge of the tool, relative to the first normal detected (usually above the selected edge).

 

 Toolpath along Normal 2: Allows you to work with the tip of the tool, relative to the second normal detected (usually below the selected edge).

 

 Toolpath along Tangent 2: Allows you to work with the edge of the tool, relative to the second normal detected (usually below the selected edge).

 

 Toolpath along median axis: Allows you to work on the axis between the 2 detected normals.

 

 

 Z axis direction: Usable with a 3-axis machine, this mode allows you to follow a non-planar profile.

 

 

 Types of trajectories : If the selected path is following the tangeante, then it is possible to follow a path according to the normal or U / V curves.

 Toolpath along normale : Mode activated by default, the path followed is normal to the profile.

 

 Toolpath along U parameter : The trajectory is followed according to the detected U curve.
 Toolpath along V parameter : The trajectory is followed according to the detected V curve.
