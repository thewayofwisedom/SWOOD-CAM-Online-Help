HELICAL INTERPOLATION

 

Introduction

The Helical Interpolation operation allows the selected profiles to be bypassed with a constant Z-axis descent. It can be used to process cylindrical profiles, whether internal or external.

 

Selection

It is available via side face selection (parallel to the programme Z-axis) cylindrical only.

Once the tool and the Helical Interpolation milling operation have been selected, the operation configuration window will appear on the screen. 

For details on the data common to all operations, see the SWOODCAM Millings operations page.

 Blind profiles :

  1. Approach Z value
  2. Maximum depth of passes
  3. Bottom oversize (positive or negative values accepted)
  4. Extra thickness lateral (positive or negative values accepted)
  5. Extra thickness total

 Through-all profiles :

  1. Approach Z value
  2. Maximum depth of passes
  3. Bottom exceeding (positive or negative values accepted)
  4. Extra thickness lateral (positive or negative values accepted)
  5. Extra thickness total

An additional approach parameter is available in this operation for lead-in :

  1.  Inside with an approach value
  2.  Inside centred
  3.  Outside with an approach value

Extension to the raw : Automatically detects the height of the profile to be processed up to the raw part. If activated, parameter 5 (Height to be processed) will no longer be displayed.

  1.  Extension to raw deactivated
  2.  Raw extension over the profile
  3.  Raw extension under the profile
  4.  Raw extension over and under the profile

 

For the creation of an automatic operation, see the section Automatic Millings.
