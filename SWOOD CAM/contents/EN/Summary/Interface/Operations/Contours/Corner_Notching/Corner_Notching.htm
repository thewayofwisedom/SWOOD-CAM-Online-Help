OPERATION: CORNER NOTCHING

WHAT IS CORNER NOTCHING?

The  Corner Notching operation is used to machine the corners of through-hole profiles, using an angled tool. This operation can be performed after a simple "Contouring" operation.

Squaring a concave angle

Operation creation

Once a machining entity and a  saw-type tool (preferably on an angled aggregate) has been chosen, simply select the "  Corner Notching" operation and "add" it.

 

OPERATION PARAMETERS

The "  Corner Notching" operation contains the following specific parameters:

 

Automatic distance to angle

Activation: By activating this  button, the operation uses the diameter of the previous tool to calculate the distances to the corner, left and right.

Deactivation: If the  button is disabled, the left and right corner distances must be entered manually, using the "Distance to angle on left side" and "Right corner distance" parameters.
