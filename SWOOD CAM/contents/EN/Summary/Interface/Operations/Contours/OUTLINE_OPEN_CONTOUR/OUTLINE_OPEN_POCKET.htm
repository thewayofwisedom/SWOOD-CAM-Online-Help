OUTLINE OPEN POCKET

 

Introduction

The Outline open pocket feature is used to machine the concave areas of part contours in the same way as pockets.

 

Selection

Warning: The Outline open pocket operation is only available in automatic (without selection).

Click  on Milling  and then Outline  and select the tool and the Outline open pocket operation to access the operation configuration window.

For details on the data common to all operations, see the SWOODCAM Millings operations page.

For details on the data common to automatic operations, see the Automatic operations page.
