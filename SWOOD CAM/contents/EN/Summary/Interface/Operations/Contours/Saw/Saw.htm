SAW

 

Introduction

Saw milling makes it possible to saw one side easily. It is available via a plane face selection.

 

Sélection

By selecting a face of a profile,  it is possible to saw it by clicking  on Sawing , or on Milling  and Saw after selecting the blade.

Once the tool and the Saw milling operation have been selected, the operation configuration window will appear on the screen. 

For details on the data common to all operations, see the SWOODCAM Millings operation page.

 

Machining parameters

In opposition to the Saw Groove operation, the Saw operation is only used on open profiles.

Profile settings

After selecting the operation, the following diagram will appear on the screen, allowing you to customize the path of the contour, using the top and side faces of the contour as a reference.

 

  1.  Approach Z value
  2.  Slit depth (if slit is activated)
  3.  Bottom exceeding (positive or negative values accepted)
  4.  Extra thickness lateral (positive or negative values accepted)

 

Extension to the raw : Automatically detects the height of the profile to be processed up to the raw part.

  1.  Extension to the raw deactivated
  2.  Extension to the raw activated

Machining methods

Rotation of the C axis by 180° : During a sawing operation, the head is always positioned outside the workpiece by default to prevent collision problems. It is possible to reverse the direction if necessary,

  1.  Reversal of the motor axis

 

Slitting parameters: By default, the sawing operation is performed in one pass. It is possible to activate the slit setting to be able to cut in 2 passes.

  1.  Saw (climb or conventional milling): By default, the sawing process is performed in one operation, depending on the selected direction of climb/conventional milling.
  2.  Slit+ Saw in both directions: Allows you to create a slit with final machining in woth directions. If climb mode is selected, the incision will be made while climbing, and sawing in conventional milling. 
  3.  Slit+ Saw (climb or conventional milling): Allows you to create an incision with blade clearance before the final sawing. If climb mode is selected, the incision and final sawing will be made while climbing.

 

Extended angle : By default, the sawing operation is calculated so that the blade cuts the selected face. However, it is possible to add a phase shift, useful for example by adding a 1mm gap on the mitre cut to make it easier to glue the pieces together .

  1.  Normal profile : Mode enabled by default, the operation will cut the selected face.
  2.  Reference profile at the bottom : The blade will remain in contact with the lower edge with the phase shift on top.
  3.  Reference profile at the top : The blade will remain in contact with the top edge, the phase shift being on the bottom of the face.

 The phase shift is to be indicated in mm, and will remain the same regardless of changes in the thickness of the part.
