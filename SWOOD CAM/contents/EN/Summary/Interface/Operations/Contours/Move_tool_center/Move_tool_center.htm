MOVE TOOL CENTER

 

Introduction

The Move Tool Center operation is available as a side face selection (parallel to the Z-axis of the program), edge selection, or sketch selection.

 

Selection

  1. By selecting a face of a profile, you can Move Center Tool a complete profile by clicking  on milling  or outline . A propagation window will appear on the screen, allowing you to manage the propagation of the selection:
    1.  Selection of the previous side: Propagates the selection to the next "leftmost" side of the selection
    2.  Selection of the next side: Propagates the selection to the next "rightmost" side of the selection
    3.  Full profile selection: Propagates the selection to all the profile.
    4.  Cancel the last selection: Cancels the previous propagation.
    5.  Validate the selection: Validates the propagation of the selection on which to apply the operation
    6.  Cancel selection: Cancels the whole selection

  1. By selecting a sketch  from the SOLIDWORKS feature manager and clicking   on milling  or outline 
  2. By selecting one or more edges from the SOLIDWORKS graphical interface and clicking  on milling  or outline 

Once the tool and the Mone tool Center operation have been selected, the operation configuration window will appear on the screen.

For details on the data common to all operations, see the SWOODCAM Millings operation page.

Reference:

  1. If a face is selected, the machining reference is the plane of the "bottom" of the profile.
  2. When selecting an edge or sketch, the machining reference is the plane of the edge or sketch, respectively.

For all types of profiles, it is possible to choose between 3 profile configuration modes:

 Through-all profile :

  1. Approach Z value
  2. Maximum depth of passes
  3. Exceeding (positive or negative values accepted)
  4. Profile height

 Blind profile :

  1. Approach Z value
  2. Maximum depth of passes
  3. Depth of the last passe
  4. Bottom oversize (positive or negative values accepted)
  5. Profile height

 Reference above the profile :

  1. Approach Z value
  2. Exceeding (positive or negative values accepted)
  3. Profile height

Extension to the raw : Automatically detects the height of the profile to be processed up to the raw part. If activated, parameter 3 (Reference on top of the profile) will no longer be displayed.

  1.  Extension to the raw disabled
  2.  Raw extension above the profile

An option is available next to the graphic area :

  1.  Reverse direction of the machining paths : Reverses the direction of the toolpath.

For the creation of an automatic operation, see the section Automatic Millings.
