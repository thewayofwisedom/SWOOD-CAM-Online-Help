OUTLINE CONTOUR

 

Introduction

The Outline Contour operation is only available in automatic mode (without selection). It allows the machining of the outer contours of the workpiece to be optimised.

In fact, it will allow the SWOOD Design edgebands to be taken into account, giving a wider choice of parameters in the outline contour milling.

Caution : In order to have the opration working correctly, the option "exclude edgebands bodies" in the Selection edition window needs to be unchecked.

Selection

As mentioned above, no selection has to be made. Click on Milling  and then Outiline  and select the tool and the Outline Contour operation to access the operation configuration window.

For details on the data common to all operations, see the SWOODCAM Millings operations page.

 

Below are the parameters relating to the operation:

  1. Approach Z value
  2. Profile height
  3. Maximum depth of passes
  4. Exceeding (positive or negative values accepted)
  5. Extra thickness lateral (positive or negative values accepted)

Options relative to the operation :

Choice of the profile to be processed :

There are several options for optimising the treatment of the outer contour.

 Machining the complete external profile : This option makes it possible to create the complete contour of the part.

 Machining the contour of the raw material :

 Machining the difference between the blank and the contour of the part :

This option allows more precise filtering of case management, and gives access to the following options :

  1.  Condition of validity : This option gives access to several filters. One or more filters can be activated simultaneously

    1.  No edgeband : This option gives access to several filters. One or more filters can be activated simultaneously

    2.  Edgeband on both sides : Automatically machines adjacent faces with two edgebands

    3.  From the edgeband : Automatically machines the sides adjacent to an edgeband if the toolpath allows the face to be machined from an edgeband. This option respects the direction of the downstroke or opposition toolpath..


    4.  To the edgeband : Automatically machines the sides adjacent to an edgeband if the toolpath allows the face to be machined towards the edgeband. This option respects the direction of the downstroke or opposition toolpath.

     

 

Extension to the raw : Automatically detects the height of the profile to be processed up to the raw part. If activated, parameter 2 (Profile height) will no longer be displayed.

  1.  Extension to the raw disabled
  2.  Raw extension enable

An option is available next to the graphic area to choose between :

  1.  Customizable lead-in : Allows you to freely choose the lead-in point by left-clicking on it. .
  2.  Lead-in in the middle of the longest segment : Allows you to lead-inin the middle of the longest segment of the profile. By activating this option, it becomes impossible to select the lead-inpoint manually via a left click.
