PARTIAL CONTOURING

 

Introduction

Partial Contour Milling operation is a standard milling that is available by side face selection (parallel to the Z-axis of the program), edge selection or sketch selection.

 

Selection

  1. By selecting a face of a profile, you can contour a complete profile by clicking  on Milling  or Outline . A propagation window will appear on the screen, allowing you to manage the propagation of the selection:

    1.  Selection of the previous side: Propagates the selection to the next "leftmost" side of the selection
    2.  Selection of the next side: Propagates the selection to the next "rightmost" side of the selection
    3.  Full profile selection: Propagates the selection to all the profile.
    4.  Cancel the last selection: Cancels the previous propagation.
    5.  Validate the selection: Validates the propagation of the selection on which to apply the operation
    6.  Cancel selection: Cancels the whole selection

  1. By selecting a sketch  from the SOLIDWORKS design tree and clicking  on Milling  or Outline  
  2. By selecting one or more edges from the SOLIDWORKS graphical interface and clicking  on Milling   or Outline 

Once the tool and the Partial Contour milling operation have been selected, the operation configuration window will appear on the screen. 

For details on the data common to all operations, see the SWOODCAM Millings operations page.

 

 

Reference :

  1. If one side is selected, the machining reference is the plane of the "bottom" of the profile.
  2. If an edge or sketch is selected, the machining reference is the plane of the edge or sketch respectively.

 

Principle

The Partial Contour allows a part of a profile to be machined. Thus, the following parameters are specific to it:

 

Choice of Profile :

 For Through-all profiles :

  1. Approach Z value
  2. Profile height
  3. Maximum depth of passes
  4. Exceeding (positive or negative values accepted)
  5. Extra thickness lateral (positive or negative values accepted)

 

 For blind profiles :

  1. Approach Z value
  2. Profile height
  3. Maximum depth of passes
  4. Depth of the last passe
  5. Bottom oversize (positive or negative values accepted)
  6. Extra thickness lateral (positive or negative values accepted)

 For profiles on edges or sketches

In face selection, the offset side of the tool is automatic. However, in the case of edge or sketch selection, it will be possible to choose the offset side in addition to the various parameters explained above.

  1.  Reverse side of the offset : Allows the offset side of the tool to be changed when machining in relation to a profile.

 

Extension to the raw : Automatically detects the height of the profile to be processed up to the raw part. If activated, parameter 2 (Profile height) will no longer be displayed.

  1.  Extension to the raw disabled
  2.  Raw extension above the profile
