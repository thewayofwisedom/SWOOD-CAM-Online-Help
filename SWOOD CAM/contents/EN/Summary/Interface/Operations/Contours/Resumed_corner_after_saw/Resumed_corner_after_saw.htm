RESUMED CORNER AFTER SAW

 

Introduction

The Resumed corner after saw  is an operation to process the inside corners of a workpiece contour, usually machined by a first Saw operation.

 

Selection

By selecting a face of a profile, you can resumed a corner by clicking  on milling  or outline . A propagation window will appear on the screen, allowing you to manage the propagation of the selection :

  1.  Selection of the previous side: Propagates the selection to the next "leftmost" side of the selection
  2.  Selection of the next side: Propagates the selection to the next "rightmost" side of the selection
  3.  Full profile selection: Propagates the selection to all the profile.
  4.  Cancel the last selection: Cancels the previous propagation.
  5.  Validate the selection: Validates the propagation of the selection on which to apply the operation
  6.  Cancel selection: Cancels the whole selection

Once the tool and the Resumed corner after saw milling operation have been selected, the operation configuration window will appear on the screen. 

For details on the data common to all operations, see the SWOODCAM Millings operations page.

 

 Through-all profiles

  1. Approach Z value
  2. Profile height
  3. Maximum depth of passes
  4. Exceeding (positive or negative values accepted)
  5. Extra thickness lateral (positive or negative values accepted)

 

 Blind profiles

  1. Approach Z value
  2. Profile height
  3. Maximum depth of passes
  4. Depth of the last passe
  5. Bottom oversize (positive or negative values accepted)
  6. Extra thickness lateral (positive or negative values accepted)

 

 Custom Profiles

By selecting this option, the following diagram will appear on the screen, allowing you to customize the path of the contour, using the top and side faces of the contour as a reference. 

  1. Extra thickness lateral (positive or negative values accepted)
  2. Approach Z value
  3. Exceeding (positive or negative values accepted)
  4. Height of the profile in relation to the upper contour limit

 

Scheme :

  1. Diameter of the roughing tool (saw)
  2. Length of the approach ramp
  3. Approach distance
  4. Disengagement distance
  5. Length of the release ramp
