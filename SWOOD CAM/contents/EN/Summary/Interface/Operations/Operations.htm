OPERATIONS

WHAT ARE THE OPERATIONS?

A machining operation consists in the definition of trajectories that a tool will take to remove material from the part.

The  SWOOD CAM software allows to convert the geometries of the parts into machining operations. There are 4 types of operations:

  1. Contours
  2. Drill
  3. Pockets
  4. Surfaces

 

ACCESS TO OPERATIONS

There are several ways to add operations to the milling phase:

  • By automatic detection: Some operations can be generated by automatic recognition of the part geometry. For this, no selection is necessary, just select the command for one of the automatic operations.

 

  • By face selection: Click on one of the faces to be machined. Depending on the selection, in the contextual toolbar, click on the " Milling", " Drilling", " Sawing" or " Clamex" button.

Then the propagation window opens to allow the user to select other contiguous faces.

Selection propagation window

  1.  Select previous face: Allows to propagate the selection to the left face of the selected face.

  1.  Select next face: Allows to propagate the selection to the face to the right of the selected face.

  1.  Selection full profile: Allows to propagate the selection to all contiguous faces of the profile.

  1. Select bottom: Allows to propagate the selection to the face below the selected face.

 

  1.  Undo previous selection: Allows to undo the last propagation.

  1.  Validate the selection: Allows to validate the propagation of the selection on which to apply the operation.

 

  1.  Cancel the selection: Allows to cancel the selection.

 

  • By sketch or edge selection: Select a sketch (open or closed) or an edge of the part, then click on the " Milling" or " Outline" button.

 

CHOICE OF OPERATION

The operation selection window allows to choose the operation to be applied. This choice is made according to the selection, the machining command and the available tool.

This interface is visible when the  machining is selected in the "List of compatible inserted tools".

3 actions are possible from this window:

  • Insert a machining from the library: Select a machining from the machining library that is compatible with the selection and click on "insert".
  • Insert an existing machining: Select an existing machining entity in the current milling phase and click on "insert".
  • Add a new operation: Click on one of the tools in the list of compatible inserted tools, select an operation (in the "New machining operations" list) and click on "Add".

This interface is visible when a tool is selected in the "List of compatible inserted tools".

 

OPERATION EDITION

Once an operation is added, it is possible to edit this operation with the following interface:

This interface is visible when an  operation is selected in the "List of compatible inserted tools".

 

The operation edition window is composed of 2 tabs:

  • The MACHINING OPERATION tab: Allows to edit the current parameters of the machining operation.
  • The TECHNOLOGY tab: Allows to edit the advanced tool and machine parameters for the operation.

 

MACHINING OPERATION tab

The MACHINING OPERATION tab is composed of 4 areas:

  • Parameters Area: Allows to set the main parameters of the machining operation.
  • Tools and operations list: Lists the inserted tools and operations.
  • Graphics Area: Allows to preview the trajectory of the operation.
  • Profiles Area: Allows to select the different profiles and to position the attachment points.

 

Parameters Area

The parameters area is composed of several types of parameters depending on the selected operation, however some parameters are often present:

Approach parameters

Depending on the selection and operation, the approach parameters may vary:

Start/end: If this parameter is available, it allows to dissociate the start approach point from the end approach point.

  Tangent Approach: Approach where the tool gradually enters the material tangent to the profile.

   Normal approach: Approach where the tool enters directly into the material perpendicular to the profile.

  Axial approach: Approach where the tool plunges into the material from the top of the part.

   Approach ramp: Approach where the tool plunges progressively into the material from the top of the part at an angle.

  Approach ramp & tangential: Approach combining the advantages of ramp approach and tangential approach.

Helical Approach: Approach where the tool plunges into the material following a helical trajectory.

 Approach plan: Approach where the tool enters directly into the material perpendicular to the profile without plunging.

Fast movements parameters

Allows to configure the machine's fast movements between two profiles:

 Security clearance Z at 1° hole, approach XY then dive Z: Default fast movement parameter, the safest one.

 Approach XY and then dive in Z: Fast movement without Z-clearance.  Faster movement setting, but with collision risks.

 Diving in Z then XY Approach:  Faster movement setting, but with collision risks.

 XYZ approche simultaneously:  Faster movement setting, but with collision risks.

Tool pass parameters

Les paramètres de passes d’outil dépendent du type d'opération éditée.

Some parameters are common to all operations:

 Z security clearance when finished: Additional clearance safety behavior at the end of the operation. Linked to the following parameter "Z Security".

Z Security: Safety height value of the tool at the end of an operation, related to the fast movement parameters.

Approach Z value: Tool clearance height value before making fast movements.

Other specific parameters are detailed in the help for each operation.

Dimensional parameters

The dimensional parameters are used to configure the operation, they are different for each operation.

Other parameters

Work mode:

  •  Downcut: The cutting tool rotates in the direction of the workpiece feed. Provides better surface finish for less machining time, but may reduce tool life.
  •  Upcut: The cutting tool rotates in the opposite direction of the workpiece feed. Provides less vibration and cutting effort, but can provide poor surface finish or even tear out wood fiber.

Reference type:

Depending on the selection and the operation, it is possible to choose different reference types:

  •  Opening profile: The milling reference plans to undercut the part, includes the "Bottom Exceeding" parameter.
  •  Not opening profile: The milling reference does not plan to cut under the part, includes the parameters of "Bottom oversize" and "Depth of the last pass".
  •  Top profile reference: The milling reference is limited to the selected edge or sketch, including the "Exceeding" parameter.

Depending on the type of selection (automatic, faces, edges, ...) the default reference will be different.

Please note that changing the reference type will have an impact on the available dimensional parameters.

 

Extension to the raw:

  •  Upper part, does not take into account the rough and the "Profile Height" is modifiable.

  •  Upper part, takes into account the raw and the "Profile Height" is not modifiable.

  •  Lower part, takes into account the raw and the "Profile Height" is not modifiable.

  • Lower and upper part, takes into account the raw and the "Profile Height" is not modifiable.

 

Tools and operations list

The tools and operations list summarizes the tools present in the active machining phase and lists all operations that belong to the active machining.

Indications on operations:

 Regular operation (the icon varies according to the chosen operation).

 Error: an error makes the operation impossible.

 Warning: one element requires the user's attention, but the operation is still possible.

Contextual toolbar:

By  right-clicking on an operation, there are the following options:

  • "Rename": Allows to assign a new name to the operation.
  • "Copy": Allows to copy the selected operation.
  • "Delete": Allows to remove the selected operation.
  • "What’s wrong?" (visible only if the operation has an error or a warning): opens a window explaining the problem(s) detected.

 

Graphics Area

The Graphics Area is used both to visualize the machining profiles, and also to configure certain parameters such as the landing point or the attachment points.

Zoom in on profile: Allows to completely hide the Parameters Area in order to better see the Graphics Area.

Zoom out on profile: Allows to show the Parameters Area again.

WHAT IS THE SPLINE APPROXIMATION?

The " Changing spline approximation" is a window where it is possible to set the decomposition of SolidWorks " Splines" into segments or arcs. This step is crucial for the post-processor to correctly program tool paths.

 

INTERFACE

The " Changing spline approximation" window is composed of 3 areas:

 

Graphics Area

The Graphics Area represents the spline(s) to be approximated and also the previous and current approximation.

 

Parameters

Several parameters can be used to modify the spline approximation:

  • "Tolerance": Maximum normal distance between the initial profile and the approximation.

  • "Minimum segment length": Length that a segment of the approximation can make, at least.

  • Two radio buttons to choose between a "Segment" or "Arc" based approximation.

 

Key

The key is used to illustrate what is represented in the graphics area, using three colors:

  • "Initial profile": Represents the spline as it was drawn in SolidWorks.
  • "Previous Approximation": Allows to compare the impact of parameter changes.
  •  "Current Approximation": Allows to visualize the impact of the parameters.

Contextual toolbar:

By  right-clicking on the graphics area, there are the following options:

Selection

  • "Selecting the landing point" (option checked by default): Allows to select by clicking the landing point of the operation on the profiles.
  • "Selecting current profile" (works only with automatic operations): Allows to select the profile to be edited from all the profiles processed by the operation.

Attachment points

  • "Selecting a point of attachment": Allows to insert attachment points on the profile.
  • "Deleting an attachment point": Allows to remove attachment points present on the profile.
  • "Edit attachment configuration"

    WHAT IS THE "EDIT ATTACHEMENT CONFIGURATION"?

    The "Edit attachment configuration" is a window where it is possible to set all dimensions of the attachment points.

     

    INTERFACE

    The "Edit attachment configuration" window is composed of 2 areas:

    Parameters

    The 3 attachment point configuration parameters represent the following dimensions:

    • " Length": Defines the total length of the attachment (at the bottom of the corners).
    • " Height": Defines the height of the attachment from the bottom of the panel.
    • " Angle": Defines the angle of the attachment from the bottom of the panel.

    Buttons

    The 3 buttons of the window:

    • "Apply all": Allows to apply the modified parameters to the already inserted and the following attachments.
    • "Ok": Allows to apply the modified parameters to the following attachments only.
    • "Cancel": Cancels the modified parameters.

Zoom

  • "Dynamic zoom": Allows to zoom in and out by holding down the  left click and moving the cursor up and down.

  • "Window zoom": Allows to zoom in on a selection frame made with the mouse.

  • "Previous zoom": Allows to return to the previous zoom.
  • "Total zoom": Allows to see all the profiles detected by the operation in the Graphics Area.
  • "Panoramic": Allows to move the display, by keeping the left click pressed and moving the cursor in the desired direction.

  • "Trajectories 3D"

Profiles Area

In the Profiles Area there are two types of commands: commands on profiles (visible only if there are several profiles in the selection) and commands for positioning the landing point:

Profiles commands

 Selection of the previous profile.

 Selection of the next profile.

Commands for positoning the landing point

There are two ways to position the landing point:

 Lead in on the longest segment: Mode that automatically places the landing point in the middle of the longest segment of the profile.

  •  Approach on the previous segment.
  •  Approach on the next segment.

Technology Tab

WHAT IS THE TECHNOLOGY TAB?

The "TECHNOLOGY" tab allows to set the machine technologies during machining, as long as they are present on the machine and controlled by SWOOD.

 

INTERFACE

The TECHNOLOGY tab is composed of 4 parameter areas:

 

Tool parameters

The tool parameters are used to set what will be displayed in the machining programs:

  1. Tool name: Display name of the tool in the SWOOD CAM tool library.
  2. Head Value: Head number that will be used.
  3. Tool Value: Tool number.
  4. Diamete: Tool diameter (in mm).

 

Suction cap and deflector parameters

  1. Suction cap position: From top to bottom, sets the position of the suction cap:
    1. 0 .
    2. 1 .
    3. 2 .
    4. 3 .
    5. Custom position,  to set manually.
  • Deflector/spoiler: Use of the deflector/spoiler during machining if the box is checked.

 

Tool corrector parameters

Allows to assign a specific marker number depending on the marker chosen (1st or 2nd).

 

Other parameters

  1. Spindle rotation direction: Allows to choose the direction of rotation of the spindle.
    1.  Spindle rotation clockwise.
    2.  Trigonometric spindle rotation.
    3.  Spindle stopped.
  2. Blow activation:
    1.  Blow inactive.
    2.  Blow active.
  3. Tool speed parameters:
    1. Tool speed: If checked, allows to use the tool parameters in the SWOOD CAM library. If unchecked, allows to customize the speed settings of the tool.
    2. Frequency: Spindle rotation frequency in rpm.
    3. Feed: Feed rate in m/min.
    4. Diving feedrate: Tool plunge speed in m/min.
    5. Angle C/X: Force angle C during machining (angle around the Z axis). Allows to force the machining with a precise angle C.
    6. Feed correction in circular interpolation: Correction of the feed rate according to the size of the arcs.
    7. Feed correction relative to the grain direction: Correction of the feed rate according to the grain direction. Feed grain correction parameters will appear on the left.
    8. Marking angles: If checked, allows to mark a pause in the corners. 
      1. Downtime in the seconds: Stopping time in the corners.
      2. Critical angle: Angle from which the option will be active.
    9. Staples: If checked, allows to add staples/clips.
      1. Reference: Reference of the staple/clip to be inserted.
      2. Maximum number of clips (1): Maximum number of staples/clips that can be inserted.
      3. Distance along the first clip (2): Distance between the first edge of the segment and the staple/clip.
      4. Distance along the last clip (3): Distance between the second edge of the segment and the staple/clip.
      5. Minimum spacing (4): Minimum distance between the axis of two staples/clips.
