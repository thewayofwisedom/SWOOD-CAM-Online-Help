OPERATION: ANGLE FINISHING

 

WHAT IS ANGLE FINISHING?

The  Angle Finishing operation is used to machine the inside of concave angles, using a conical milling cutter.

To be effective, this operation is generally used after a "Resume pocket angles" or "Resume angles using radius compensation" operation.

OPERATION PARAMETERS

The  "Angle finishing" operation contains the following specific parameters:

 

Side edge length

Size at which machining starts; depends on the diameter of previous tools.

 

Transitions

Transitions represent rapid tool paths between two parts to be machined. There are two possibilities:

Diagonal transition

Zigzag transition

Straight transition between two angles to be machined.
