RESUME ANGLE USING RADIUS COMPENSIATION

 

Introduction

The Resume angle using radius compensation is an operation that allows you to process the corners of geometries present in a part, generally machined by a Pocket milling, coded with the tool radius compensation.

 

Difference with Resumed pocket angles:

Resumed pocket angles

Resume angle using radius compensation

 

Selection

  1. By selecting the bottom of a pocket, it is possible to finish the walls of a pocket by clicking  on Milling .
  2. By selecting a sketch  in the SOLIDWORKS FeatureManager tree and clicking  on Milling .
  3. By selecting a closed outline from the SolidWorks graphical interface and clicking  on Milling .
  4. By selecting a closed loop (edges) from the SolidWorks GUI and clicking  on Milling .

 

Once the tool and the operation Resume angle using radius compensation have been selected, the operation configuration window will appear on the screen.

For details on data common to all operations, see the SWOODCAM Milling page.

 

For through profiles

This includes sketch , closed contour or closed loop (edge) selections.

  1.  Approach Z value
  2.  Profile height
  3.  Maximum depth of passes
  4.  Bottom oversize (positive or negative values accepted)
  5.  Extra thickness lateral (positive or negative values accepted)

 

Milling after routing / Cut-groove milling:

  1.  Milling after routing: Allows to process the corners after have used a pocket milling.

After have created a Pocket milling, thi smilling detect automaticly the tool previously used to calculate the remaining paths. However, it is also possible to specify the diameter of the previous tool manualy.

  1. Approach distance
  2. Disengagement distance
  3. Diameter of the roughing tool

 

  1.  Milling after cut-groove milling: Allows to process the corners after have used a Cut-groove milling.

 

  1. Approach distance
  2. Diameter of the roughing tool
  3. Disengagement distance

 

For blind profiles

  1.  Approach Z value
  2.  Profile height
  3.  Maximum depth of passes
  4.  Depth of the last passes
  5.  Bottom oversize (positive or negative values accepted)
