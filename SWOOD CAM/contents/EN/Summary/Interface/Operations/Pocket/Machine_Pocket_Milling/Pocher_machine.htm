MACHINE POCKET MILLING

 

Introduction

The Machine Pocket milling is an operation for processing the blind geometries (without islands) present in a part. Contrary to the Pocket Milling, this operation does not export a trajectory but allows you to export a recess macro with the parameters indicated in SWOOD CAM.

This operation is available for export to the following software:

  • BiesseWorks
  • BSolid
  • IMAWop
  • Maestro
  • TPA

 

Selection

  1. By selecting the bottom of a pocket, it is possible to machine a pocket by clicking  on Milling .
  2. By selecting a sketch  in the SOLIDWORKS Feature manager tree and clicking  on Milling .
  3. By selecting a closed outline from the SolidWorks graphical interface and clicking  on Milling .
  4. By selecting a closed loop (edges) from the SolidWorks GUI and clicking  on Milling .

Once the tool and the Machine Pocket have been selected, the operation configuration window will appear on the screen.

 

For details on data common to all operations, see the SWOODCAM Milling page.

For details on the data common to automatic operations, see the Automatic operations page.

 

For the through profiles

This includes sketch selections , closed contour selections or closed loop selections (edges).

  1.  Approach Z value
  2.  Maximum depth of passes
  3.  Bottom oversize (positive or negative values accepted)
  4.  Extra thickness lateral (positive or negative values accepted)
  5.  Profile height

 

Extension to the raw : Automatically detects the height of the profile to be processed up to the raw part. If activated, parameter 5 (Profile Height) will no longer be displayed.

  1.  Extension to the raw deactivated
  2.  Extension to the raw activated
 

For blind profiles

  1.  Approach Z value
  2.  Maximum depth of passes
  3.  Depth of the last passes
  4.  Bottom oversize (positive or negative values accepted)
  5.  Extra thickness lateral (positive or negative values accepted)
  6.  Profile height

 

Extension to the raw : Automatically detects the height of the profile to be processed up to the raw part. If activated, parameter 5 (Profile Height) will no longer be displayed.

  2.  Extension to the raw activated
