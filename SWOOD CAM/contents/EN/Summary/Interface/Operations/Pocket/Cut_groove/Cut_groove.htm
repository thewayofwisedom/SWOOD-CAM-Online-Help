CUT-GROOVE

 

Introduction

The Cut-Groove operation is an operation to process the hollow geometries present in a part using a saw.

 

Selection

  1. By selecting the bottom of a pocket, it is possible to machine a pocket by clicking  on Milling .
  2. By selecting a sketch  in the SOLIDWORKS Feature manager tree and clicking  on Milling .
  3. By selecting a closed outline from the SolidWorks graphical interface and clicking  on Milling .
  4. By selecting a closed loop (edges) from the SolidWorks GUI and clicking  on Milling .

Once the tool and the Cut-Groove milling have been selected, the operation configuration window will appear on the screen.

For details on data common to all operations, see the SWOODCAM Milling page.

 

Through-all profiles:

  1.  Approach Z value
  2.  Maximum depth of passes
  3.  Exceeding (positive or negative values accepted)
  4.  Machining depth
  5.  Tool thickness %
  6.  This is a non-changeable parameter indicating the detected groove width.

 

Multipass with Jump between passes or Zig-Zag :

  1.  Mill with jump between passes : Allows the grooves to be machined by repositioning the blade between each pass.
  2.  Mill without jump between passes : Allows the grooves to be machined without repositioning the blade between each pass. The climb/conventional direction varies according to the passes.

 

Extension to the raw : Automatically detects the height of the profile to be processed up to the raw part. If activated, parameter 4 (Machining depth) will no longer be displayed.

  1.  Extension to the raw deactivated
  2.  Extension to the raw activated

Blind profiles:

  1.  Approach Z value
  2.  Maximum depth of passes
  3.  Machining depth
  4.  Tool thickness %

 

Multipass with Jump between passes or Zig-Zag :

  1.  Mill with jump between passes : Allows the grooves to be machined by repositioning the blade between each pass.
  2.  Mill without jump between passes : Allows the grooves to be machined without repositioning the blade between each pass. The climb/conventional direction varies according to the passes.

 

Extension to the raw : Automatically detects the height of the profile to be processed up to the raw part. If activated, parameter 3 (Machining depth) will no longer be displayed.

  1.  Extension to the raw deactivated
  2.  Extension to the raw activated
