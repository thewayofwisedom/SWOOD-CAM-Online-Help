POCKET MILLING

 

Introduction

The Pocket milling is an operation to process the cavities geometries present in a part.

 

Selection

  1. By selecting the bottom of a pocket, it is possible to machine a pocket by clicking  on Milling .
  2. By selecting a sketch  in the SOLIDWORKS Feature manager tree and clicking  on Milling .
  3. By selecting a closed outline from the SolidWorks graphical interface and clicking  on Milling .
  4. By selecting a closed loop (edges) from the SolidWorks GUI and clicking  on Milling .

Once the tool and the Pocket Milling have been selected, the operation configuration window will appear on the screen.

For details on data common to all operations, see the SWOODCAM Milling page.

 

For the through profiles

This includes sketch , closed contour or closed loop (edge) selections.

  1.  Approach Z value
  2.  Maximum depth of passes
  3.  Bottom oversize (positive or negative values accepted)
  4.  Extra thickness lateral (positive or negative values accepted)
  5.  Profile height

Multipass with Jump between passes or Zig-Zag :

  1.  Mill with jump between passes : Allows the pockets to be machined by moving up in Zsecu after each pass.
  2.  Mill without jump between passes : Allows the machining of pockets without going up in Zsecu. Between each pass the tool plunges without moving upwards.

 

Extension to the raw : Automatically detects the height of the profile to be processed up to the raw part. If activated, parameter 5 (Profile Height) will no longer be displayed.

  1.  Extension to the raw deactivated
  2.  Extension to the raw activated

For blind profiles

  1.  Approach Z value
  2.  Maximum depth of passes
  3.  Depth of the last passes
  4.  Bottom oversize (positive or negative values accepted)
  5.  Extra thickness lateral (positive or negative values accepted)
  6.  Profile height

Multipass with Jump between passes or Zig-Zag :

  1.  Mill with jump between passes : Allows the pockets to be machined by moving up in Zsecu after each pass.
  2.  Mill without jump between passes : Allows the machining of pockets without going up in Zsecu. Between each pass the tool plunges without moving upwards.

 

Spiral path with constant Z / jumping between disjoint zones :

  1.  Spiral path with constant Z : Allows the machining of pockets with disjointed areas without repositioning the tool between the areas.
  2.  Spiral path with jumping between disjoint zones : Allows the machining of pockets with disjointed zones by moving the tool upwards between the different zones..

 

Extension to the raw : Automatically detects the height of the profile to be processed up to the raw part. If activated, parameter 5 (Profile Height) will no longer be displayed.

  1.  Extension to the raw deactivated
  2.  Extension to the raw activated
