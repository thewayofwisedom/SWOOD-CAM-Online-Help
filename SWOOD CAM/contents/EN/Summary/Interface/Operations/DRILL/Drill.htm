DRILL

 

Introduction

The Drill operation allows you to drill holes, or cylindrical partial holes. It can be used with several types of tools.

 

Selection

In order to create a drilling operation, a hole or a partial hole (cylindrical surface) must be selected. After selecting the geometry to be drilled, click on  a hole, then click   on the drilling icon . The window for selecting identical holes:

  1.  Select all: Allows you to select all holes of the same depth and diameter as the selected one.
  2.  Deselect all: Allows you to deselect all holes of the same depth and diameter as the selected one.
  3.  Validate the selection: Validates the selection on which to apply the operation.
  4.  Cancel selection: Cancels the selection.

It is possible to select only some of the identical holes via the SOLIDWORKS GUI.

Once the tool and the Drill operation have been selected, the operation configuration window will appear on the screen.

Once the tool selected, it will be possible to choose between different operations :

  1. Drill
  2. Drill reaming
  3. Drill-chips breakers
  4. Drill-Breaking
  5. Helical Interpolation
  6. Contour
  7. Pocket milling
  8. Tap (only if a "Tap" tool is available and selected)

After selecting the desired operation, the operation setting window will appear on the screen.

For details on the data common to all operations, see the SWOODCAM Millings operation page.

The operations Drill, Drill reaming, Drill-chips breakers, Drill-Breaking have common options that enable them to be set up correctly.

Drill operation options:

Drill reaming :

The operation Drilling reaming is distinguished by its additional parameter "Delay" which keeps the tool at the bottom of the hole for the time declared in this parameter. The delay time is in seconds.

It is possible to machine drilling chamfers as a drilling operation with a chamfering cutter.

 Use hole diameter to chamfer (default value)

This parameter needs to be set to drill regular hole.


The graphic area :

The graphic area is empty if the tool used is a simple tool (because the choice of tool was made before choosing the operation).

If a drilling block is used, the graphic area will display the drilling block diagram.

It will therefore be possible to select the spindle(s) to be used when drilling.

  1. The spindles used are displayed in blue 
  2. Unused spindles are displayed in grey 

 Note: If several spindles are selected, when drilling, they will all be used in multi-spindle descent (if managed by the machine), even if there is no geometry to be drilled under certain spindles.

To the right of the graphic area there is an additional option :

  1.  Reverse boring direction : Allows the drilling direction to be reversed. The orientation of the spindle will also be reversed.

Options for the Helical interpolation milling :

It is possible to choose to treat the selected geometry as :

 Blind geometry :

  1. Approach Z value
  2. Maximum depth of passes
  3. Bottom oversize (positive or negative values accepted)
  4. Extra thickness lateral (positive or negative values accepted)
  5. Extra thickness total

 Through-all geometry

  1. Garde d'approche en Z
  2. Profondeur maxi des passes
  3. Dépassement en fond (valeurs positives ou négatives acceptées)
  4. Surépaisseur Latérale (valeurs positives ou négatives acceptées)
  5. Profondeur d'usinage

An additional landing parameter is available in this operation for lead-in:

  1.  Interioir with value
  2.  Interior center
  3.  Exterior with approach value

 

Options for the Contour milling :

Il est possible de choisir de traiter la géométrie sélectionné en tant que géométrie

 Blind geometry :

  1. Approach Z value
  2. Maximum depth of passes
  3. Depth of the last passes
  4. Bottom oversize (positive or negative values accepted)
  5. Extra thickness lateral (positive or negative values accepted)
  6. Extra thickness total

 Through-all geometry :

  1. Approach Z value
  2. Extra thickness total
  3. Maximum depth of passes
  4. Bottom oversize (positive or negative values accepted)
  5. Extra thickness lateral (positive or negative values accepted)

 Personalized Profile

By selecting this option, the following diagram will appear on the screen, allowing you to customise the path of the contour, using the top and side faces of the contour as a reference.

  1. Extra thickness lateral (positive or negative values accepted)
  2. Approach Z value
  3. Exceeding (positive or negative values accepted)
  4. Height of the profile in relation to the lower contour limit

Options for the Pocket Milling :

Depending on the type of profile selected, different options will be available:

Through-all geometry :

  1.  Approach Z value
  2.  Maximum depth of passes
  3.  Bottom oversize (positive or negative values accepted) 
  4.  Extra thickness lateral (positive or negative values accepted)
  5.  Extra thickness total

Multipass with Jump between passes or Zig-Zag :

  1.  Mill with jump between passes : Allows the pockets to be machined by moving up in Zsecu after each pass.
  2.  Mill without jump between passes : Allows the machining of pockets without going up in Zsecu. Between each pass the tool plunges without moving upwards.

 

Extension to the raw : Automatically detects the height of the profile to be processed up to the raw part. If activated, parameter 5 (Profile Height) will no longer be displayed.

  1.  Extension to the raw deactivated
  2.  Extension to the raw activated

Blind geometry :

  1.  Approach Z value
  2.  Maximum depth of passes
  3.  Depth of the last passes
  4.  Bottom oversize (positive or negative values accepted)
  5.  Extra thickness lateral (positive or negative values accepted)
  6.  Profile height

Multipass with Jump between passes or Zig-Zag :

  1.  Mill with jump between passes : Allows the pockets to be machined by moving up in Zsecu after each pass.
  2.  Mill without jump between passes : Allows the machining of pockets without going up in Zsecu. Between each pass the tool plunges without moving upwards.

 

Extension to the raw : Automatically detects the height of the profile to be processed up to the raw part. If activated, parameter 5 (Profile Height) will no longer be displayed.

  1.  Extension to the raw deactivated
  2.  Extension to the raw activated

Options for Tap Milling :

As previously stated, the Tap milling is only available by using a "Tap" tool. Depending on the type of profile selected, different options will be available:

 Through-all geometry :

  1.  Approach Z value
  2.  Bottom oversize (positive or negative values accepted) 
  3.  Profile height

 Blind geometry :

  1.  Approach Z value
  2.  Bottom oversize (positive or negative values accepted) 
  3.  Profile height
