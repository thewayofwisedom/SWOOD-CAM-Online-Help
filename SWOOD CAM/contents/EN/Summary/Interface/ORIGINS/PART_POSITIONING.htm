ORIGINS

 

WHAT ARE ORIGINS?

In SWOOD CAM, a part's  origin defines its position on the machine, and thus influences the coding of machining operations.

 

HOW TO CREATE AN ORIGIN?

There are several ways to create an origin:

  • From the menu bar: Go to Tools / SWOOD CAM /  "Components positionning".
  • From the SWOOD CAM CommandManager: Click on the  "Components positionning" button.
  • From the SWOOD CAM toolbar: Click on the  "Components positionning" button.

 

INTERFACE

Once the  "Components positionning" command has been selected, the "Positionning" PropertyManager opens on the left.

 

Interface for a part

The origins PropertyManager interface for a part is divided into 3 sections:

  • Position
  • Stock
  • Program origins

 

Position

In SWOOD CAM, there are 3 types of positioning:

  1.  Based on coordsys selection
    •  : Defines the reference origin.
    •  : Defines X axis direction.
    •  : Defines Y axis direction.
    •  : Defines Z axis direction.

Sketch points or vertices, edges or sketch lines, or reference faces or planes can be selected to define orientation.
Note also that defining two axes will automatically define the third to create a direct trihedron.

 Reverse buttons are used to reverse the direction of the concerned axis.

  1.  Based on automatic recognition

It is possible to automatically define the part orientation based on a pre-defined positioning strategy.

  • Strategy: Defines positioning strategy.
    • Click on the drop-down list to choose a positioning strategy.
    • Click on  Edit to modify the list of positioning strategies (help on positioning rules).
  • Reference body: Defines the reference body on which the strategy is to be applied, according to the selected machining strategy (help on positioning rules).
  • Program top: Sets the panel's Z axis orientation.
    •  Panel top: Selected by default, orients the top of the panel to the face containing the most machining operations.
    •  Panel bottom: Orients the bottom of the panel to the side containing the most machining operations.
  1.  Based on SWOOD Design panel

The orientation of the part can be defined by using the orientation defined in SWOOD Design.

  • Program top: Permet de définir l'orientation de l'axe Z du panneau.
    •  Panel top: Selected by default. Orient the Z axis normal to the top face of the panel.
    •  Panel bottom: Reverse the Z direction if the default orientation is not correct.
  • Program front edge:
    •  Front edgeband: Allows part positioning with front edgeband defined as program front edgeband.
    •  Right edgeband: Allows part positioning with right edgeband defined as program front edgeband.
    •  Back edgeband: Allows part positioning with back edgeband defined as program front edgeband.
    •  Left edgeband: Allows part positioning with left edgeband defined as program front edgeband.

 

Stop

List of stops available on the machine (if controlled by SWOOD CAM). Use the drop-down menu to choose which stop will be used for part positioning.

 

Offset

Allows parts to be offset on the machine along the X, Y and Z axes defined previously.

  • The  panel icon can be used to select the stock as the reference for calculating offsets.
  • The  origin icon can be used to select the SOLIDWORKS origin as the reference for calculating offsets.

 

Stock

In this section, it is possible to select the stock data for the part.

With  selection-based positioning, it's possible to manually modify stock values for thickness, length and width.

If the stock values are different from those of the part, the surplus (or deficiency) will be distributed evenly on both sides of the part.

  • Exclude hidden bodies: Allows hidden bodies to be ignored in the stock definition.

 

In the positioning mode  based on SWOOD Design panel, the following parameters are available:

  1. Use stock extension: Takes into account the stock extension defined in SWOOD Design.
  2.  Use panel with edgebands: Takes edgebands defined in SWOOD Design into account. Valid if edges are glued before machining.
  3.  Include top laminate: Takes into account the laminate applied to the top surface as defined in SWOOD Design. Valid if the laminate is applied before machining.
  4. Include bottom laminate: Takes into account the laminate applied to the bottom surface as defined in SWOOD Design. Valid if the laminate is applied before machining.

 

Program origins

  • Creating origins

It is possible to define a program origin different from OP0.

By default, the  OP0 origin is already present in the list.

To create a new origin, it is necessary to :

  1. Click on the  origins button on the right of the list.
  2. In the "Selection" tab below, in the  zero box, select a point to define the point of origin of the inclined plane.
  3. In the "Selection" tab below, in the Z axis box, select an axis or a plane face to define the Z axis direction of the inclined plane.
  4. It is possible to reverse the direction of the Z axis by clicking on the  reverse button.

In the "Values" tab, there are several important information about the selected inclined plane:

  • Angle A: (cannot be modified) Defines angle A measured in relation to OP0.
  • Angle C: (cannot be modified) Defines angle C measured in relation to OP0.
  • X: (cannot be modified) Defines the offset value of the inclined plane origin from OP0 along the X axis.
  • Y: (cannot be modified) Defines the offset value of the inclined plane origin from OP0 along the Y axis.
  • Z: (cannot be modified) Defines the offset value of the inclined plane origin from OP0 along the Z axis.
  • Name: Defines the name of the inclined plane, as it will appear in the list of origins.
