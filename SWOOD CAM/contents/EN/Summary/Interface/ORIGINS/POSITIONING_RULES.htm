POSITIONING RULES

 

Introduction

SWOOD CAM's positioning rules can be defined to be used and allow the automatic positioning of a part according to certain criteria.

 

Library configuration

  1.  Allows you to create a new positioning rule.
  2.  Allows you to save changes made to the positioning rules.
  3.  Allows you to delete the selected positioning rules.

Click on a positioning rule to display the following parameters:

  1. Generalities
    1.  Name: Designation of the positioning rule.
    2.  Description: Description of the positioning rule.

 

  1. Positioning Criteria
    1.  Body selection: Allows you to choose a reference body that will be used as a template to apply the rule.
      1. Biggest body: Automatically chooses the application body to the ruler. The choice of the body will be grayed out and managed automatically.
      2. User selection: Allows you to manually choose the volume body on which to apply the positioning rule.
    2.  X choice: Sets the orientation of the X-axis on the panel.
      1. Minimize panel width: Sets the Z-axis to the smallest dimension of the panel, and by default sets the X-axis to the length of the panel.
      2. Longest groove: Sets the X-axis in the direction of the largest groove in the panel.

        If this option is selected, groove recognition options will be visible.

 

  1. Groove recognition parameters (only available when "X choice" is set to "Longest groove")
    1.  Min length: Minimum groove length to be taken into account by the positioning rule in mm.
    2.  Max thickness: Maximum panel thickness supported by the detection algorithm in mm.
    3.  If no recognized groove: Behavior if no groove is recognized. This is an informative parameter that cannot be modified, always set to "Minimize panel width".
    4.  Groove Orientation: Defines the groove to be recognized for positioning and groove orientation.
      1. Indifferent: The groove will have no impact on positioning.
      2. Downwards: The longest groove is positioned at the bottom of the panel in relation to the origin.
      3. Upwards: The longest groove is positioned at the top of the panel in relation to the origin.
