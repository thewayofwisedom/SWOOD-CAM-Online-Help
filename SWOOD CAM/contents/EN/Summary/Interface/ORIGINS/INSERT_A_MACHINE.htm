INSERT A MACHINE

It is possible to implement a table plan representing a machine with suction cups or pressers and to adapt the position of the latter to the part/assembly to be machined.

 

Open the machine edition tab

Right clic  on " No machine" which is under " Positionning", then clic on " Insert machine"

A "PropertyManager" tab called "Machine" will open

 

Choose a machine to position

Click on the drop-down list under "Name" and choose the machine you want to position

Note: The files of the machines to be positiionned are located in SWOODData/Plans

To delete a machine positionning, clic on " Delete"

 

Load a machine state

Once the machine is selected, you will see one or more "states" in the "States" area.

Select a state, clic on " Load machine state", and clic on "Yes"

To delete a machine state, clic on " Delete all positionning of table"

Note: This deletion will not delete the saved machine states

 

Create a new machine state

To create a new machine state, there are two possibilities:

  • Using the table editing window

           To do this, clic on " Modify machine state"

  • Directly on SolidWorks graphic view

           To do this, complete the following steps:

Clic on a suction cup  or a clamp 

 

You can make a translation of this element by holding the click and moving the mouse

 

You can rotate this element by holding the click on the circle above the element and moving the mouse

You can switch the type of element from the "Suction cups / clamps" list

Once the element is modified you can change all the other elements of the table at once by clicking on " Set the current element on all supports".

 

Once the table position has been modified, you can save this machine status by clicking on " Save table positionning".
