SWOOD CAM OPTIONS

SWOOD Options are available on Tools → SWOOD CAM →  Options.

This windows espacially allows to choose the data repertory needed to manage the diferent libraries of SWOOD Design or SWOOD CAM. 

It also allows to set some options available in SWOOD CAM.

To save the configuration of SWOOD CAM and to apply it SOLIDWORKS need to be close and open again.

Main

 Data directory   Allows to define the user data repertory folder for SWOOD Design : the SWOOD Data.
 Working directory Allows to define the folder where the working files are saved by default when there are copied.
 Language

Allows to choose the language to use in the SWOOD feature. By default the language used is the same as the SOLIDWORKS language. [-use SOLIDWORKS Language]
 Debug mode Activate/Desactivate the debug mode
 XML counters directory Allows to define a repertory for the counter during the generation of XML files.
 Report configuration file Allows to define another .cfg file for the report configuration

Units

  Use metric system    Allows to choose if SWOOD use the metric system or the imperal system.
  Hide units symbol Allows to display or to hide the units symbol in the SWOOD windows.

Report

  Report configuration file Defines an alternative report configuration file to that of SWOOD Data.
System Report Enables or disables items in the system report.
 Show Report Errors during generation Select whether or not to display errors during report generation.

Design

This part is a SWOOD CAM specific part. Here there are all the options according to the different type of elements. 

Treeview settings                                          

 Show properties at bottom The designation of the program and the used postprocessor is displayed under the machining treeview of SWOOD CAM.
 Label edit allowed Allows the renaming in the SWOOD CAM machining tree of the different informations linked to the tools and machinings.

 

Custom operation names: Allows to create customized machining operation names based on filters, from the Customized operation names window.
