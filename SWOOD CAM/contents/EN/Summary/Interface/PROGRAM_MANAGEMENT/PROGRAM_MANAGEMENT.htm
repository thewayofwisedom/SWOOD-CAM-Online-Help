PROGRAM SETTINGS

WHAT IS PROGRAM SETTINGS?

 Phase management is used to create one or more milling phases (programs) on a SolidWorks part or assembly. This allows to define the targeted machine, as well as specific global machining parameters.

 

ACCESS THE PROGRAM SETTINGS WINDOW

There are several ways to access the program settings window:

  • From the menu bar: Go to Tools / SWOOD CAM /  Program settings.
  • From the SWOOD CAM CommandManager: Click on  Program settings button.
  • from the SWOOD CAM toolbar: Click on  Program settings button.

 

INTERFACE

The Program settings window is divided into 2 sections:

  • The Phase_list.
  • The Parameters section.

 

Sections

Phase list

Milling phase creation: Above the phase list, there is a  button for creating a new milling phase.

Milling sequence: It is possible to rearrange the order of milling phases in the list. To do this, select a phase in the list, then click on the  up arrow to move the phase up, or on the  down arrow to move it down.

 

Parameters section

The parameters section consists of several tabs: Machine, Information, Nesting and Feed/Grain correction.

  1. Machine: Defines parameters related to the program to be coded and the machine.
    •  Name: Defines the name of the milling phase as it will appear in SWOOD CAM's FeatureManager.
    •  Program numben: Defines the program number as it will appear in the coded program (useful for some post-processors).
    •  Machine: Selects the post-processor that will be used to encode the program. The list depends on the post-processors installed in the SWOODData "PostProcessors" folder.
    •  Report settings: Options when the program is encoded from the SWOOD report. 
      • No machining to encode alert: Select how to warn when a program contains no machining.  The "Show Report Errors during generation" option must be checked in SWOOD CAM Options.
        • None: In this case, when a report is generated, there are no errors either during or after generation.
        • Error: In this case, when a report is generated, there is an error on the loading bar during generation. There are several errors after generation: in the general variables, and in the part program.

 

  1. Information: Defines additional (optional) parameters adding information to the program. 
    • Operater: Operator name.
    • Assembly: Assembly or project name.
    • Part: Part name.
    • Reference: Part reference.
    • Number of parts: Number of identical parts.
    • Comment: Additional comments on the part.

 

  • Nesting: Defines some useful parameters when the part is used in a Nesting assembly.
    • Nested Part: Defines the current milling phase as the default phase to be brought up in the nesting program.
    • Tool diameter: Defines a custom tool diameter for nesting this part. Enables correct calculation of spacing between adjacent parts.

 

  • Feed/Grain correction: Defines additional parameters when the part has a grain direction.
    • From SWOOD Design: Loads grain parameters from SWOOD Design material and applies speed corrections defined in the material.
    •  Correction enabled: Enables or disables correction of machining speeds according to machining direction.
    •  Material: SOLIDWORKS material applied to the panel.
    •  Predefined settings: Allows to create or load correction angle and correction factor parameters. By hovering the mouse over the area adjacent to the parameter name, it is possible to:
      1. Click on the grey area next to the parameter, or on to  choose from the predefined created parameters.
      2. By clicking on Edit , the Feed/Grain correction Library window appears, allowing to add additional correction configurations to the list:
        1.  Allows to create a new correction configuration.
        2.  Allows to delete the selected configuration.
        3.  Allows to create a copy of the selected configuration.
          You can set the following parameters:
          1. Name: Configuration name.
          2. Factor: Speed correction factor.
          3. Angle: Angle between grain direction and feed direction from which correction will be applied.
    •  Correction angle: Angle between grain direction and feed direction from which correction will be applied.
    •  Correction factor: Speed correction factor when machining enters the corrected zone.
    •  Grain direction: Defines the direction of the panel grain. By clicking on the area adjacent to the parameter, it is possible to:
      • The  tab lets you define angles manually, by entering angles around the X, Y and Z axes, or by defining an Alpha and Beta angle, which correspond to rotation around the Z and Y axes respectively.
      • The  tab lets you define a custom fiber direction using the SolidWorks model. You can select an edge, a sketch line, a face or a plane.

Please note that the grain direction will be:

  • Depending on the direction of the edge or sketch line.
  • Normal to the selected plane or face.
