PROGRAM CONFIGURATION

It si possible to configure the program suppression state according to different parameters

 

Reach the program configuration window

To reach the program configuration window, right clic  on the program and clic on  "Configure"

 

Program configuration parameters

The program configuration window "Program validity" contains two SolidWorks parameters that can turn the program into a suppression state:

  • According to a SolidWorks configuration
  • According to a Custom Property
