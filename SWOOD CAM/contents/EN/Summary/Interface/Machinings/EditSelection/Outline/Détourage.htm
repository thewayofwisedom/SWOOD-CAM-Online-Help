EDIT SELECTION: OUTLINE

It is possible to edit the selection of the machining " Outline" by right-clicking on the machining.

 

INTERFACE

The selection editing window is composed of several parameters.

 

PARAMETERS

Body selection

It is possible to select which bodies will be impacted by the machining:

  • "All bodies": All the bodies of the model will be impacted by the machining.
  • "Selected bodies": Only the bodies selected in the selection box will be impacted by the machining.

 

Edgebands

It is possible to exclude SWOOD Design edgebands from the machining.

  • With the "Exclude edgeband bodies" option checked.

  • With the "Exclude edgeband bodies" option unchecked.
