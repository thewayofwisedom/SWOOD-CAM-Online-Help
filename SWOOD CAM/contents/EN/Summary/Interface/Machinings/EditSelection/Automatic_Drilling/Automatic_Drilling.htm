EDIT SELECTION: HOLES

It is possible to edit the selection of the machining " Holes" by  right-clicking on the machining.

 

INTERFACE

The selection editing window is composed of several parameters.

 

PARAMETERS

Body selection

It is possible to select which bodies will be impacted by the machining:

  • "All bodies": All the bodies of the model will be impacted by the machining.
  • "Selected bodies": Only the bodies selected in the selection box will be impacted by the machining entity.

 

Selection filter

It is possible to exclude some holes from the machining with the following selection filters:

  • "Do not drill holes drilled by a previous auto-drilling": Does not take into account holes already drilled by an automatic drilling.
  • "Do not drill holes drilled by a manual drillng": Does not take into account holes already drilled manually.
  • "Complete cylinders only": Only takes into account holes defined by a complete cylinder.
  • "Define partial cylinder min angle": Defines the angle of the partial cylinders that will be taken into account in the drillings. Possibility to exclude the entered angle by checking the "Strict" box.
