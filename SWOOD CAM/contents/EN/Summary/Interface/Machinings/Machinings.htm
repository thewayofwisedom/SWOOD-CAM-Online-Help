MACHININGS

WHAT IS A MACHINING?

A machining entity allows to group one or several machining operations of the same type.

 

It also allows to save the parameters of these operations in the machining library by drag and drop.

 

CREATE A MACHINING

There are 4 ways to create a machining:

  1. From the SolidWorks Menu Bar:
    1. Click on "Tools".
    2. Click on "SWOOD CAM".
    3. Select the desired machining.

  1. From the CommandManager:
    1. Select the "SWOOD CAM" tab
    2. Select the desired machining.

 

  1. From the SWOOD CAM tab toolbar:
    1. Open the  SWOOD CAM tab on the left pane.
    2. Either click on the "Milling" command, then select the desired milling machining.
    3. Either click on the "Drilling" command.

  1. From the 3D model geometry:
    1. In the Graphics Area, select one of the faces of the model.
    2. In the window that opens, depending on the selected face, one or more machinings appear.

 

Once the machining command is launched, the machining editing window opens.

It is then possible to select one of the tools present in the milling phase, then add one of the available operations.

 

EDIT A MACHINING

Editing a machining is useful for modifying the parameters of operations contained in the machining and for creating new operations.

 

To edit an existing machining, it must be in a milling phase.

Simply  right-click on the concerned machining, then click on " Edit".

Then the machining editing window opens, so it is possible to edit the machining or the operations it contains.

 

CONFIGURE A MACHINING

Configuring a machining allows to both delete or activate the entity and to create a machining pattern.

 

To configure an existing machining entity, it must be in a milling phase.

Simply  right-click on the concerned machining and click on "Configure".

Then the Machining configuration window opens and it is possible to configure it.

 

EDIT SELECTION

Editing the selection of a machining allows to adjust some detection parameters and filters in order to obtain an adapted machining.

 

To edit the selection of an existing machining, it must be in a milling phase.

Simply  right-click on the automatic machining in question, then click on "Edit Selection".

It is possible to edit the selection of only certain machinings:

  • The  Outline machining.
  • The  Holes machining.
  • The  Auto groove machining.
  • The  Auto pocket machining.
  • The  Auto flat face machining.
  • The  Auto Clamex machining.
  • The  Surface machining.
