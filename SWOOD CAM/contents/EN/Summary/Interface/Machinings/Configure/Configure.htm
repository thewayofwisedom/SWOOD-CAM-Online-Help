CONFIGURE MACHININGS

WHAT IS THE MACHINING CONFIGURATION?

The machining configuration allows several things:

It allows to activate or not the machining according to a  SolidWorks Configuration, a  Feature or several  Features.

It also allows to be repeated through a link to a linear or circular SolidWorks pattern, thanks to the " Pattern" tab.

 

HOW TO CONFIGURE A MACHINING?

To configure an existing machining, it must be in a Milling phase.

Simply  right-click on the desired machining entity and click on "Configure".

 

INTERFACE

The configuration window is divided into 2 tabs:

  • The machining  Suppression/Validity tab.
  • The machining  Pattern tab.

 

TABS

Suppression/Validity tab

The machining Suppression/Validity tab is composed of 4 tools:

 Configuration

The machining will be deleted if one of the checked configurations is activated.

 

 Feature

The machining will be deleted if the selected SolidWorks feature is in  suppression state.

 

 Features

The machining will be activated only if it meets the condition that depends on one or more SolidWorks functions.

 Custom property

Pattern tab

The  Pattern tab allows to associate the machining with a SolidWorks pattern function.

 

 The machining operation must be created in the base geometry of the pattern. If another geometry is selected, the pattern will be generated from this geometry and the result will not be correct.
