TOOL EDIT FROM SWOOD CAM TREE

 

Introduction

The Tools management tab is shown after : 

  •  double click on a current phase tool.
  •  right clicked on a current phase tool and then  Edit 

 

The tab allos to visualize and to modify the selected tool paramters. These parameters have been set in the tool library.

By default it's only possible to visualize the parameters. 

To have the possibility to modify them and so to modify the current phase tool it's possible to delete the link with the library. 

To do so extent the Library source section and  unticked the parameter   Linked with library. After that the parameter becomes available to modify them.
