TOOLS MANAGEMENT

 

Introduction

The Tools Management  window is accessible from the SWOOD CAM programming tree, from the SWOOD feature bar, or from the menu  "Tools > SWOOD CAM > Tools". 

To find help on the interface topic, click here.

This window allows you to insert or replace tools in the targeted programming phases.

 

Tools management

After to have click  on the Tool Management , a new configuration window will appear on the screen.

 

This window is divided into three areas, from left to right you have :

  • Left area: The program list with the inserted tools.
    These buttons can be used:
    •  Copy tool: Copies the selected tool.
    • : Delete tool: Deletes the selected tool.
  • Central area: The settings window.
    Displays the parameters of the selected entity in the left area (program or tool information).
    • After clicking on a current phase tool all its parameters are displayed. These parameters are set in the tools library. 
      But it also possible to delete the library link to allow modification on them only on the current phase. 
      To do so extent the Library source section and  unticked the parameter   Linked with library. After that the parameter becomes available to modify them. 
  • Right area: Tool libraries.
    From this area it is possible to:
    • Display standard tool libraries.
    • Display aggregates library
    • Add the selected tool to the current program
    • Replace the tool selected in the current program with the tool selected in the library.
