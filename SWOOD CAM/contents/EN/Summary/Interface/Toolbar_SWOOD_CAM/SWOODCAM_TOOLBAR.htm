SWOOD CAM TOOLBAR

 

The toolbar or the SWOOD CAM tab contains all the main SWOOD CAM features.

 

  • To display the SWOOD CAM tab in the SOLIDWORKS CommandManager :
    • When a file is open  Right-click on the SOLIDWORKS CommandManager
      → Tabs
      → Select SWOOD CAM
    • When a file is open  Right-click on the SOLIDWORKS CommandManager
      → Customize... 
      All available tabs are then displayed under the SOLIDWORKS toolbar. Those with a black title are displayed and those with a grey title are hidden.
      →  Right click on the SWOOD CAM toolbar
      → Display the tab

 

  • To display the Toolbar :
    • From the Display options of SOLIDWORKS:
      View → Toolbars →  SWOOD CAM
    • From the SOLIDWORKS CommandManager:
       Right click → Toolbars  →  SWOOD CAM

 

By default the toolbar and the tab are composed of 9 main features:

  1.  Program settings
  2.  Components positionning
  3.  Tools management
  4.  Milling
  5.  Drilling
  6.  3D Milling
  7.  Outline
  8.  Sawing
  9.  Auto Groove
  10.  Auto Pocket
  11.  Clamex
  12.  Simulate
  13.  Report
