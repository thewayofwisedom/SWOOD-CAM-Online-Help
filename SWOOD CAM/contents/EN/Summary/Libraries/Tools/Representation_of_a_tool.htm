TOOL REPRESENTATION

 

Introduction

In order to use specific tool profiles with Material Removal Simulation, it is necessary to create a tool profile.

To access the profile declaration section, simply edit the desired tool in the library, then click in the Simulation section and then Profil.

 

Interface

The following interface will appear on the screen to configure the simulation mode as also to activate it.

The left side corresponds to the tool representation.

The right side allows to:

  • File : Select the file corresponding to the template.
  • Sketches : If several sketches are present in the model, allows you to select the desired one.
  • Origins : If several origins are present in the model, allows you to select the desired one.
  • Vertical flip : Flips the sketch vertically.
  • Horizontal flip : Flips the sketch horizontally.

The cutting part of the tool appears in red, depending on the useful length of the tool entered in the library.
