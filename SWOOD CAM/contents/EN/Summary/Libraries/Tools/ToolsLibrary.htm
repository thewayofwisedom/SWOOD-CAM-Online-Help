TOOLS LIBRARY

 

Introduction

The SWOOD CAM Tools Library  is accessible from the SWOOD CAM library  in the SOLIDWORKS task pane. It lists all the tools created by the user.

 

 

Edit Tools Library

To create a tool, right click  in the SWOOD CAM tool library, then select "Edit". The tool libraries configuration window will appear on the screen.

 

 

Management

  •  Allows you to create :
    • A new library of tools if "Tools Libraries" is selected.
    • A new tool if "the previously created tool library" is selected.
  •  Allows you to delete the selected item

By right-clicking  on a tool library, or on a created tool, it is possible to:

  •  Create a new tool.
  •  Delete the selected item.
  •  Copy the selected item.
  •       Rename the selected item.
  •  Create a new folder (Available only by selecting a folder or library).

To configure a tool :

  • Fill in the geometrical data of it by completing the appropriate zones. Each type of tool has parameters.
  •  Complete the standard elements, common for all tools :
    • Name : Tool designation. Defines the display name of your tool in the SWOOD CAM tool library.
    • Tool type : Defines the type of tool used.
    • Reference : Defines the tool reference.
    • Head : Defines which CNC head will be used.
    • Tool number : Defines the tool number.
  • Correctors :
    • Corrector 1 : Tool corrector.
    • Corrector 2 : Corrector n°2 of the tool. Available only with some tools.
  • Cutting Settings :
    •  Rotation : Defines the direction of rotation of the tool.
      •  Clockwise/Right : The tool rotates clockwise.
      •  Counter-clockwise/left : The tool rotates in the trigonometric/counter-clockwise direction.
      •  Arrêt : The tool does not rotate.
    • Spindle speed : Tool rotation frequency in rpm.
    •      Feedrate : Tool feed rate in m/min.
    •      Feedrate down : Penetration feed rate of the tool in m/min.
  • Suction hood :
    •  Position : Definition of the position of the suction hood during machining with the tool.
      •  None : Unused hood.
      •  High : Hood in high position.
      •  Middle : Hood in intermediate position.
      •  Low : Hood in low position.
      •  Custom : Hood in custom position.
    • Height : Customized height of hood. This option is only available with the "Custom" hood position.
  • Miscellaneous :
    •  Deflector : Enables or disables the deflector when using the tool.
    •  Blower : Enables or disables the blower when using the tool.
  • Default milling parameters : These parameters change the default values in the operations where the tool is used.
    •  Milling cut : Defines the machining mode of the tool.
      •  Downcut : Downcut mode.
      •  Upcut : Upcut mode.
    •  Depth cut Max : Maximum depth of cut achievable by the tool.

Additional parameters are available when the tool "Drill Thread Milling Cutter" is selected.

  • Default milling parameters : Ces paramètres modifient les valeurs par défaut dans les opérations où l'outil est utilisé
    •  Milling cut : Définit le mode de travail de l'outil.
      •  Downcut : Mode de travail en avalant.
      •  Upcut : Mode de travail en opposition.
    •        Threading feedrate : Reglage de l'avance du filetage en m/min
    •  Depth cut Max : Profondeur de passe maximum réalisable par l'outil.
    •  Threading Entry Feed Ratio : Ratio de la vitesse d'entrée par rapport à la vitesse de plongée.
    •       Threading Diameter : Diamètre du filet en mm
    •  Threading Entry Type :

      • 180 : The tool makes a circular arc in order to land tangent to the surface.

      • Direct : The tool lands normally on the surface

    •  Threading Z Offset from thread bottom : Décalage en Z du filetage
    •  Drilling step : Pas de perçage de l'outil en mm

  • Simulation :
    • Profil : Allows you to load a customized profile and associate it with the tool. This profile will be loaded during the material removal simulation. See Tool representation
