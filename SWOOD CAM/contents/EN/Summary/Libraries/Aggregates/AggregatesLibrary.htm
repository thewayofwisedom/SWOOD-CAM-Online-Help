AGGREGATES LIBRARY

 

Introduction

The SWOOD CAM aggregates library  can be accessed through the SWOOD CAM library   from the SOLIDWORKS task pane. It lists all the aggregates created by the user.

 

 

Edit the aggregates library

To create an aggregate, right click  in the SWOOD CAM aggregates library, then select "Edit". The Aggregate Libraries configuration window will appear on the screen.

This window is divided into three sections:

  • The Aggregate Management Tree
  • The graphic area
  • The settings area

 

 

The Aggregate Management Tree

  •  Allows you to create a new aggregate.
  •  Allows you to delete the selected item.

By right clicking  on the aggregate library   , it is possible to:

  •  Create a new aggregate.
  •  Allows you to create a new folder.

By right clicking  on an aggregate , it is possible to:

  •  Create a new aggregate.
  •  Delete the selected item.
  •  Copy the selected item.
  •       Rename the selected item.

By right clicking  on a floder, it is possible to:

  •  Create a new aggregate.
  •  Delete the selected item.
  •       Rename the selected item.
  •  Allows you to create a new folder.

 

 

The graphic area

The graphics area is a section allowing you to view and manage your aggregate (adding / removing spindles).

  •  Allows you to add a new pin to your aggregate.
  •  Used to copy the selected spindle.
  •  Used to delete the selected spindle.
  • Allows you to zoom in completely on your aggregate.

You can move around in the graphics window in the same way as in SOLIDWORKS. In general, we have:

  • Click of the wheel : Allows you to rotate around the aggregate
  • CTRL + Click of the wheel  :  Allows you to move the aggregate
  • Wheel rotation   : Zoom in / out

By right-clicking  in the graphic area, you can :

  • Add a new spindle to your aggregate.
  • Fully zoom in on your aggregate.

By left-clicking  it is possible to select an element in the graphic area.

 

 

The parameter area

This zone is used to manage according to the selection:

Managing the Global Parameters of the aggregate

By clicking on the gray part in the graphics area, you will be able to modify the global parameters of your aggregate.

Global parameters of the aggregate

  • Generalities
     
    •  Name: Name of the aggregate as it will be displayed in the program.
       
    •  Aggregate type: Type of aggregate to be configured:
       
      • Drilling block: The drilling block is an aggregate which contains drilling tools (bits, drills, ...)
         
      • Horizontal spindle: A horizontal spindle is an aggregate which contains all types of tools, and which allows specific operations to be carried out.
         
    •  Autodefined: Defines an aggregate that automatically creates spindles only for detected holes.  Warning: all holes will be detected (even the large cylindric pockets or internal contours).

    •  Tool number: Used to name the tool in the program (based on the machine’s model).
       
    •  Head: Default head number (for machines using multiple heads).
       
  • Rotation
     
    •  Change frequency: Authorizes or not the frequency modification for each spindle.
       
      • Globally editable: The rotation frequency remains modifiable from a tool, The frequency will be adjusted on all the other tools.
         
      • Individually modifiable: The rotation frequency can be changed from the tool. The frequency of the other tools will remain unchanged.
         
      • No changeable (by default): The frequency of the tools present in the aggregate will take the value of the rotation frequency defined in the aggregate, and will not be modifiable.
         
    •  Spindle speed: Rotation frequency of tools mounted on the aggregate (in rpm)
       
    •  Gear ratio: For some machines, the spindle speed is not directly controlled by the motor but by a gear system. It is necessary to apply a gear ratio factor to obtain the spindle frequency.
       
    •  Rotation direction: Defines the tool’s rotation direction.

      • Clockwise modifiable: Defines the rotation direction of the aggregate spindles clockwise, but remains modifiable on the spindles.
         
      • Clockwise no modifiable:  Defines the rotation direction of the aggregate spindles clockwise, and cannot be modified on the spindles.
         
      • Counter-Clockwise modifiable: Defines the rotation direction of the aggregate spindles in Trigo or counter-clockwise, and remains modifiable on the spindles.
         
      • Counter-Clockwise no modifiable: Defines the rotation direction of the aggregate spindles in Trigo or counter-clockwise, and cannot be modified on the spindles.
         
  • Geometry:
     
    •  X offset: Additional offset of your aggregate along the X axis in mm
       
    •  Y offset: Additional offset of your aggregate along the Y axis in mm
       
    •  Z offset: Additional offset of your aggregate along the Z axis in mm
       
  • Properties
     
    •  Move toolpath from inclined plane to OP0: Choice of coordinate system for operations on inclined plan.
       
      • No faces: Encoding the tool’s coordinates in the program’s main origin except for faces.
         
      • Cancel: No decomposition
         
      • Never: Encoding the tool’s coordinates in the Inclined plan specific origin (see figure)
         
      • Always: Encoding the tool’s coordinates in the program’s main origin OP0 (see figure)

 

 

  • Rotary axis
     
    • C axis:
       
      •  C axis type: 
         
        • Fixed: Rotation around the C axis is not possible
           
        • Continuous rotation C: Rotation around the C axis is possible continuously
           
        • Indexed head: Autorizes only the listed values
           
        • Twist head: Authorizes all values between minimum value and maximum value

 

With the selection of a horizontal spindle you will have one additional setting available in this section.

  • B axis:
     
    •  B axis type: 
       
      • Fixed: Rotation around the B axis is not possible
         
      • Continuous rotation B: Rotation around the B axis is possible continuously
         
      • Indexed head: Autorizes only the listed values
         
      • Twist head: Authorizes all values between minimum value and maximum value

By defining the aggregate as a drilling block, the following section will be available:

Axis limits

  1.  X Min: Limit in the X- direction of the point X = 0 of the aggregate.

  2.  X Max: Limit in the X+ direction of the point X = 0 of the aggregate. 

  3.  Y Min: Limit in the Y- direction of the point Y = 0 of the aggregate. 

  4.  Y Max: Limit in the Y+ direction of the point Y = 0 of the aggregate.

 

 

Management of selection parameters

By selecting an element in the graphical interface, it is possible to modify its parameters.

By selecting a spindle in a drill block, its spindle options will be displayed.

 Drill block spindle options

  1. Spindle
     
    •  Number: Number of the spindle.

    •  Activation: Activation ID of the spindle.

    •  Rotation: Defines the rotation direction of the spindle.
       
      •  Clockwise/Right: The spindle turns clockwise.
         
      •  Counter-clockwise/Left: The spindle turns counter-clockwise.
         
      •  Stop: The spindle does not turn.
         
  2. Geometry
     
    •  Type: Defines the type of spindle
       
      • Horizontal: The spindle will be horizontal
         
      • Vertical: The spindle will be vertical
         
    •  X offset: Offset of the spindle from point 0 of your aggregate along the X axis in mm
       
    •  Y offset: Offset of the spindle from point 0 of your aggregate along the Y axis in mm
       
    •  Z offset: Offset of the spindle from point 0 of your aggregate along the Z axis in mm
       
    •  Orientation: Defines the orientation of the spindle


                For horizontal spindles :
       
      • X-:  The orientation of the spindle axis is along X-
         
      • X+: The orientation of the spindle axis is along X+
         
      • Y-: The orientation of the spindle axis is along Y-
         
      • Y+: The orientation of the spindle axis is along Y+


        For vertical spindles :
         
      • Z-:  The orientation of the spindle axis is along Z-
         
      • Z+:  The orientation of the spindle axis is along Z+
         
  3. Tools
     
    •  Tool type: Defines the tool type used.
       
    •  Name: Set the display name of your tool in the SWOOD CAM tool library.

    •  Reference: Defines the reference of the tool.
       
    •  Corrector 1: Corrector of the tool.
       
    • Spindle speed: Tool rotation frequency in rpm.

    •      Feedrate: Tool feedratein m / min.
       
    •      Feedrate down: Tool feedrate down in m / min
      .
  4. Default milling parameters: These parameters modify the default values in operations where the tool is used.
    •  Depth cut maw: Maximum depth of cut achievable by the tool.

By selecting a spindle in a horizontal spindle, those spindle options will be displayed.

 Horizontal spindle options 

  • Spindle
     
    •  Number: Number of the spindle.

    •  Activation: Activation ID of the spindle.

    •  Rotation: Defines the rotation direction of the spindle.
       
      •  Clockwise/Right: The spindle turns clockwise.
         
      •  Counter-clockwise/Left: The spindle turns counter-clockwise.
         
      •  Stop: The spindle does not turn.
  • Geometry
     
    •  X offset: Offset of the spindle from point 0 of your aggregate along the X axis in mm.

    •  Y offset: Offset of the spindle from point 0 of your aggregate along the Y axis in mm.
       
    •  Z offset: Offset of the spindle from point 0 of your aggregate along the Z axis in mm.
       
    •  B0 angle: Defines the orientation of the spindle according to angle B.
       
    •  C0 angle: Defines the orientation of the spindle according to angle C.

    •  
  • Tools
     
    •  Tool type: Defines the tool type used.
       
    •  Name: Set the display name of your tool in the SWOOD CAM tool library.

    •  Reference: Defines the reference of the tool.
       
    •  Corrector 1: Corrector 1 of the tool.

    •  Corrector 2: Corrector 2 of the tool.
       
    • Spindle speed: Tool rotation frequency in rpm.

    •      Feedrate: Tool feedrate in m/min.
       
    •      Feedrate down: Tool feedrate down in m/min.

  • Default milling parameters: These parameters modify the default values in operations where the tool is used.

    •  Depth cut maw: Maximum depth of cut achievable by the tool.
