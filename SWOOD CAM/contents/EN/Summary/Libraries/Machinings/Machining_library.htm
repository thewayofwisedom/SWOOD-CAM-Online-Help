MACHINING LIBRARY

Introduction

The SWOOD CAM machining library is accessible from the SWOOD CAM library  in the SOLIDWORKS task pane. It lists all the machinings saved by the user.

 

 

Creation of a machining entity

To create a machining , simply create an operation and drag and drop the corresponding machining into the machining library on the right.

 

 

Editing the library of machining

It is possible to edit the machining entity library by right-clicking on it  then select "Edit". The window for configuring the libraries of machining will appear on the screen.

This window is divided into four sections:

  • Library edition toolbar
  • Machining management tree
  • Composition of the machining
  • Settings area

 

 

The library edition toolbar

Machining management tree

In this area you can see all the saved machinings.

You can filter the existing machinings with the filter bar

Right clicking  on the “ Machining library” button, you can create folders in which you can store the machinings.

Right clicking  on any  Machining, you can delete it.
