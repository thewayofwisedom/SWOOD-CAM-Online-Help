BOARDS LIBRARY

WHAT IS BOARDS LIBRARY?

The  board library allows to create and edit all Nesting boards.

To nest a part, SWOOD will look for a board with the same material as the part:

 BOARD QUANTITY: If the board quantity is not large enough to nest all the parts, some parts will not be nested. So if the quantity is 0, the plate will not be used in the nesting project.

 WRONG THICKNESS: If the panel is not the same thickness as the board material, the part will not be nested.

 

ACCESS THE BOARDS LIBRARY

To access the  boards library, you must first access the Nesting projects window (tools / SWOOD CAM /  Nesting). Then click on the  Boards Library button in the toolbar.

 

INTERFACE

The boards library window is divided into 3 sections:

  • The Toolbar.
  • The Boards list.
  • The Parameters section.

 

Sections

Toolbar

The Boards Library toolbar consists of 6 commands:

  •  New: Allows to create a new board in the board library.
  •  Save: Allows to save the existing board library.
  •  Delete: Allows to delete the selected board.
  •  Extended properties: Allows to access the extended properties window and add custom properties to the boards.
  • Import boards: Allows to import a board library (file extension *.xml) from the Windows Explorer.
  •  Help: Allows to open the help on the board library.

 

Boards list

Levels

The list of boards is organized in several levels:

  1.  Library: First mandatory level, allows to group together different boards.
    1.  Folder: Second optional level, allows to organize more finely the boards in the library.
    2. Board: Unitary element of the list.

 

Options

By  right-clicking on one of the items in the list of boards (library, folder, board) it is possible to access the following commands:

  •  New: Creates a new board in the selected item (library or folder).
  •  New folder (library or folder only): Creates a new folder in the selected item.
  •  Delete (Folder or board only): Allows to delete the selected item.
  •       Rename (Folder or board only): Allows to rename the selected item.
  •  Copy (Board only): Allows to copy the selected board.

 

 It is possible to move an item from the boards list by selecting it and dragging it to another library or folder.

 

Parameters section

If the  Boards Library is selected, the parameters displayed will be general parameters for all plates:

Generalities

  •  Name: Defines the name of the boards library.
  •  Description: Defines additional information to the boards library.

 

Reusable offcut

The reusable offcut parameters applied to the boards library will be applied to all boards in case there is no specific strategy defined in a board.

  • Number of offcuts: Defines the maximum number of reusable offcuts per board in the nesting.

  •  Minimum width and length of offcut: Defines the minimum size at which a reusable offcut can be generated.

  • Minimum surface of offcut: Defines the minimum surface (in mm²) at which a reusable offcut can be generated.

  •  Tool diameter: Defines the diameter of the tool used to machine reusable offcut only.

 

If one of the  Boards is selected, the parameters displayed will be specific to the selected board:

 

Generalities

  •  Type: Defines the type of board.
  •  Name: Defines the name of the board.
  •  Description: Sets additional information to the board.
  •  Reference: Defines a reference to the board if needed.
  •  For Board Estimation : Include this board in the boards estimation calculation in the system report.

 

Dimensions

  •  Length [X axis]: Defines the board length.
  •  Width [Y axis]: Defines the board width.
  •  Thickness [Z axis]: Defines the board thickness. If greyed out, the parameter is linked to a SWOOD Design material [See Materials section below].

 

Others

  •  Quantity: Defines the number of boards available for nesting.
  •  Cost: Defines the unit cost per board (used to calculate the project price in the report).

 

Materials

  •  Materials: Defines the SOLIDWORKS material given to the board. The name must be the same as the part for it to be used in the nesting calculation.
  •  Materials: Defines the SWOOD Design material given to the board. This can be selected by clicking on  to open the SWOOD Design material library.
  •  Grain is vertical: Defines whether the SWOOD Design material is set up with a grain, by checking the box the grain direction rotates 90° (from horizontal to vertical).
  •  Top laminate: Defines a superior laminate (SWOOD Design laminate) to the plate. The laminate can be selected by clicking on  to open the SWOOD Design material library.
  •  Top grain direction is vertical: Defines whether the SWOOD Design top laminate is set up with a grain, by checking the box the grain direction rotates 90° (from horizontal to vertical).
  • Bottom laminate: Defines a lower laminate (SWOOD Design laminate) to the board. The laminate can be selected by clicking on  to open the SWOOD Design material library.
  •  Bottom grain is vertical: Defines whether the SWOOD Design bottom laminate is set up with a grain, by checking the box the grain direction rotates 90° (from horizontal to vertical).

 

Supplier

  •  Supplier: Définit le nom du fournisseur de la plaque. Peut être utilisé pour renseigner un ERP par exemple.
  •  Supplier Reference: Defines the reference of the board at the supplier. Similar to the board reference.

 

Nesting

  •  Part types: Allows to select directly from the board a holding strategy thanks to the part types.
  •  Nesting type: Allows to select the board usage between Nesting, BeamSaw Cutting Optimization (in the report) or both.
  •  Corner: Defines the corner of the board from which the nesting starts.

  •  Direction: Defines the direction of the nesting parts on the board.

  •  Offset: Defines the distance from the edge of the board to place the parts during nesting. Like the image below, the offset is set for all directions. This is useful when the edges of the panel are not usable, such as a raw edge on a laminate.

  • Default priority: Defines the priority of the board over the others if several options are available. The higher the number, the more priority the board will be used.
  •  Not allowed machines: Defines the machine(s) that will not be able to machine the board.

 

Reusable offcut

  • Use a specific strategy: Allows to define parameters different from the general parameters of the reusable offcut.
  •  Number of offcuts: Specific parameter to the selected board, identical to the general parameters.
  •  Minimum width and length of offcut: Specific parameter to the selected board, identical to the general parameters.
  • Minimum surface of offcut: Specific parameter to the selected board, identical to the general parameters.
  •  Tool diameter: Specific parameter to the selected board, identical to the general parameters.

 

Extended properties

The extended properties are defined in the  Properties Definition window and will appear for all boards in the library. These properties can be useful in developing a custom report.

 

Machinings

  •  Enable: Enables automatic machining insertion as soon as the board is used. Once the box is checked, the following automatic machinings can be assigned and ordered:  Holes,  Groove,  Pocket,  Outline,  Offcut,  Clamex.

To define machinings, click on the list on the right, where it is written "Disable".

For each machining, it will be possible to select all the same types of machining in the " Machining Library".
Machinings can be rearranged by dragging the "  " machining button and releasing it at the desired position.
