EXTENDED PROPERTIES

The Extended Properties (or Property Definition) window is available by clicking on the Extended properties button in the Board library window.

 

WHAT ARE EXTENDED PROPERTIES?

The extended properties of the board library offer the possibility to add additional properties to the already existing board properties.

 

ADDING EXTENDED PROPERTY TO THE BOARD LIBRARY

To add an extended property just fill in the attributes of an empty line (line with an asterisk *):

  1. Name: Name of the extended property alias.
  2. Type: Property type among the following:
    1. String: The property value will be text.
    2.  Integer: The property value will be an integer.
    3. Decimal: The property value will be a decimal number.
    4.  Yes/No: The property value will be a boolean, which means either a true or a false value.
  3. Default value: Value that the property will take by default if it is not modified.
  4. Description: Displayed name in the board library.

 

EFFECT OF EXTENDED PROPERTIES IN THE BOARD LIBRARY

When the extended properties have been filled in, simply click on the validation button in the window and the extended properties will appear in the list of board properties in the Extended properties category.

 

MODIFICATION OF EXTENDED PROPERTY VALUES

In order to edit the value of an extended property for a given board, simply check the checkbox to the left of the property, which will make its value editable.
