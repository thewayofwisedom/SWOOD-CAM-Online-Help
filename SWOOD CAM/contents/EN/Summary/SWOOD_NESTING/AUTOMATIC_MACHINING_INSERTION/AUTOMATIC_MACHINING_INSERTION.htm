AUTOMATIC MACHINING INSERTION

This feature lists all the machining saved by the user in the SWOOD CAM machining library, which can be linked to the different post-processor defined in the general setting.

 

Interface

The Automatic machining insertion window consists in 2 options and 2 areas:

Options:

  • "Save": To save the machine configuration set
  • "Help": To reach the help of this topic

 

Areas:

  • List of "Machines": The list of existing postprocessors saved in the SWOODData
  • List of "Nesting" machinings:
    • The order of machining insertion can be changed with the  3 bar icon
    • The machining can be selected with the dropdown list (from the existing machinings)

 

Effect on Nesting assembly

Once the "Automatic machining insertion" saved with a machine, make sure you check the option "Enable automatic machining insertion" and selected the good machine in the Machining parameters in the Machining parameters of the Nesting projects window parameters.

After validating the nesting result, any Nesting created with this machine will insert automatically these machinings into the milling phase.
