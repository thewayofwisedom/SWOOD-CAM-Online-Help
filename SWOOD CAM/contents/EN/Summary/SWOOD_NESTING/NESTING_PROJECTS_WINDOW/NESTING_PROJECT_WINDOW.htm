NESTING PROJECTS WINDOW

The  nesting projects window is the main window of SWOOD Nesting.

 

WHAT IS SWOOD NESTING?

SWOOD Nesting is a SWOOD CAM add-in.
It allows the efficient nesting of project parts on one or more sheets, ready to manufacture.

Thanks to this add-in, parts position is optimized and waste is reduced.

ACCESS TO SWOOD NESTING PROJECTS WINDOW

There are two ways to access the main SWOOD Nesting window:

  • Either you want to create a new nesting project: 
    In the SOLIDWORKS' menu bar, click on Tools → SWOOD CAM →  Nesting

 

  • Or you want to modify a current nesting project : 
    In the SWOOD CAM tab of the project, click on   Program settings  and then on  Nesting .

 

INTERFACE

The  nesting projects window is composed of 3 main sections and 10 tools:

 

Toolbar

  •  New: Create a new nesting file
  •  Open: Open an existing document
  •  Save: Saves the active nesting
  •  Save as: Saves the active nesting in a specific folder
  •  Rebuild: to calculate or recalculate the nesting project. After every parameter change, the nesting project needs to be recalculated.
  •  Manual nesting: To access to the manual nesting [ Manual nesting help]
  •  Automatic machining insertion: To automaticaly insert machinings to the nesting results [ Automatic machining insertion]
  • Part types: To improve as much as possible the holding of the parts on the nesting boards [ Part types]
  •  Board library: To access to the board library [ Boards library help]
  • Help / Licence manager: To open the Help / To manage (Activate; transfer; show)  the SWOOD Nesting licence. If no licence is activate, SWOOD will not calculate the nesting project. 

 

Section

  • CONTENTS SECTION

The contents section is the tree structure of Nesting project:

 Nesting project assembly 

                       → Highest Nesting project level. By clicking on it you can access the nesting parameters in the parameters section. [ Nesting projects parameters help]

 

      Source files
                       → Folder containing the added files to the Nesting project.
                   → With a right click you can add a file to the Nesting project through a Windows Explorer window. The source file can be a SolidWorks part (.sldprt) or a SolidWorks assembly [.sldasm].
                   → By selecting one of the source files you can access the specific parameters of the file in the parameters section.

           Assembly
                        → Assembly added in the current nesting project
                             → With a right click on the assembly you can either " Delete" the file from the current Nesting project, or access the " Edit Fronts" function.
                             → By selecting an assembly you can access the specific parameters of the assemblies in the parameters section:
                                                      - The Quantity of this assembly to be nested.
                                                      - The path of the  File
                                                      - The  Configuration of the assembly file to use.
                                                      - The possibility for all the parts of this assembly to be  Grouped on the same board.
                                                                                  - The " Group type" allows you to choose if you want the group members to be all "On one sheet" or distributed "Over all sheets".
                                                                                  - When the "Member status" is set to "Nest" (when the group type is "On one sheet") the part will be included on another board even if there is no space on the group board. On the other hand, when the status is set to "No nest", this part will not be included.

                 Part
                        → Part added in the current nesting project
                             → With a right click on the part you can " Delete" the file from the current Nesting project.
                             → By selecting a part, you can access the same specific parameters as for assemblies in the parameters section, plus one parameter:
                                                      - Define the selected part as a  Filler Part.

Once Rebuilt, each source file in the Nesting project will be displayed in a separate color in the content section and in the results section.

 

    Parts for nesting
                       → Folder containing the parts that will be nested.
                   → By selecting this folder you will see all the parts of the source files and their information (designation, Quantity, Material and Thickness) and the possibility to activate or deactivate some parts, in the Parameters section.

               Part for nesting
                       → Single part or from an assembly that will be nested
                   → With a right click you can either " Desable" a part or " Add to sources" in order to manage this part independently from an assembly or a group (for example to change the quantity of this part only).
                   → By selecting a part to nest, you can access the Nesting parameters specific to this part, in the parameters section,[ Nesting parameters help] in addition to some specific parameters:
                                                      - "Use stocks" allows you to decompose the part to be nester by layer.

Each material layer of a part is nested on a board of the corresponding material.


                                                      - " Sketch outline" is a function that allows you to use a sketch present in the part as an outline to be used for nesting.


                                                      - The "Priority" of a part to be nested is determined by a number, the larger the number, the more the part will be nested first.


                                                      - The "Maximum quantity" of a part to be nested limits the number of copies a part can have.


                                                      - The " Flip Top/Bottom" function allows you to place a part to be nested on its other side to nest it.


                                                      - "Manual labeling" (if the label function has been activated in the nesting parameters), allows to simply move the label in the results section.

Then, when you rebuild the Nesting project, you will see the label of the modified part (and its copies) positioned in its new location.

 

                                                      - " Ignore grain direction" (editable only if the part's material has grain direction), allows to ignore the grain direction of the part, so that it can be oriented as best as possible during nesting. Can be used for parts that won't be visible, such as hidden backgrounds or side panels.

 

    Groups
                → List of groups created from the "Grouped" option of the source files.

               Group
                      → Source file defined as a group.
                   → By selecting a group you can access the group parameters in the parameters section:
                                                      - The group " Name": the name it will have.
                                                      - The "Quantity" of group that will be nested (parameter not modifiable in this interface).
                                                      - The " Group Type", see definition above.

                     Group part
                      → Source file defined as a group member.
                   → By selecting a group part, you can see many parameters in the parameters section. However, only one parameter can be modified:
                                                      - The " Member Status ", see definition above.

 

   Boards
                       → List of boards that will be used to nest the Parts for nesting. 
                   → By selecting the list of boards you will see all the required boards and their information (Name, Material, Size, Quantity, Cost, Machine and Priority) as well as the possibility of activating or deactivating certain boards, in the parameters section.

               Board
                       → Board used to nest Parts for nesting.
                   → By selecting one of the boards in the board list you can see many parameters of the board from the board library. You can even modify some of them, including several exclusive parameters:
                                                      - The "Forced Quantity", which forces a quantity of this board for this Nesting project, even if this quantity is different from the quantity in the board library.
                                                      - The " Machine" that will be used in the machining phase.
                                                      - The ability to nest from " Multiple Corner":

Option

Explanation

Example

Disabled

 The algorithm will follow its usual logic starting from the selected corner

Flipped corners

The algorithm will position the first part in the selected corner,
then the second one in the opposite corner according to the secondary direction (Y in the example)
then the next part is placed in the main direction (X in the example)

Four corners The algorithm will position the parts in descending order in the four corners of the board.

Inverted corners

 The algorithm will position the first part in the selected corner,
then the second one in the opposite corner according to the main direction (X in the example)
then the next part is placed in the secondary direction (Y in the example)


                                                      - The possibility to select a " Part Type" from the Part type configuration (if at least one part type has been set up).

   Nesting result
                       → Nesting result of the current Nesting project after a reconstruction.
                   → With a right click you can:
                                                      - Apply Nesting: Reconstruction of the Nesting project.
                                                      -  Edit nesting: Access to the manual nesting window.

               Nesting sheet
                       → Nesting result on a particular board
                   → By selecting a Nesting sheet you can see different information in the parameters section, information about the board and information about the resulting nesting scrap:
                                                      - The surface (in mm²) used by the parts.
                                                      - The remaining area (in mm²) of scrap.
                                                      - The dimension (in mm) in X of the scrap.
                                                      - The dimension (in mm) in Y of the scrap

 

  • PARAMETERS SECTION

The parameters section shows a lot of information depending on the item selected in the contents section.

The main parameters of the Nesting project are accessible when the Nesting Project Assembly is selected.

 To learn more about these parameters, go to the main Nesting parameters page.

 

  • RESULT SECTION

The result section is a 2D interface of the nesting results, so it is necessary to do a first Reconstruction to see the nested parts.

In this interface it is possible to zoom in on the board(s) by  scrolling the mouse

Display

By default, the name of the parts to be nested on the nesting sheets is taken from the SolidWorks part file name.

 

 

Visual options

When the  Nesting Result is selected in the contents section there are several visual options available:
                                                      - " Show visual identification numbering", shows the visual identification number on each part to be nested, this number is only present in the Nesting interface, it is the order found in the list of parts to be nested.
                                                      - " Use custom property name" displays on each part the value of the custom property "Description" of each part.
                                                      - " Show lead in/out part, bridges and labeling" displays both the lead in/out part and bridges, which are specific to the nesting sheet machining (which can be set in the Part Type window and in the Manual Nesting window). The labels are generated from the labeling parameters.
