NESTING ASSEMBLY

WHAT IS NESTING ASSEMBLY?

The Nesting assembly is the result of the optimization of the SWOOD Nesting algorithm, added to the modifications made through Manual nesting.

The Nesting assembly is a new SolidWorks Assembly generated by SWOOD Nesting from all the Parts for nesting in the Nesting projects window.

 

 

 

ACCESS THE NESTING ASSEMBLY

Once the Nesting result is satisfying, simply click on the validation button in the Nesting projects window (bottom right), the window will close and the SolidWorks assembly will appear.

 

NAVIGATION BETWEEN SHEETS

If the nesting result contains several sheets, simply go to the  SWOOD CAM tab to switch from one result sheet to another by double clicking on the desired milling phase.

 

MILLING

A milling phase will be created for each sheet of the nesting, with the machine selected in the Machining parameters ( Default CN).

If machinings have been set in the Automatic machining insertion window, they will be present in the milling phases of each sheet.

Otherwise, it will be possible to insert machinings in the regular way with SWOOD CAM.
