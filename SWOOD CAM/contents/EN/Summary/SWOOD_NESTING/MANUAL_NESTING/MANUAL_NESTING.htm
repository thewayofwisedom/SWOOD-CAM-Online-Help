MANUAL NESTING

Manual Nesting (or the Nesting Editor) can be reached by clicking on the  Manual Nesting button on the toolbar of the Nesting Projects Window.

Manual Nesting allows to manually modify the nesting result computed by the SWOOD Nesting algorithm.

The manual Nesting window is composed of 3 main sections, 4 tools and 2 options:

 

 

Toolbar:

  •  Moving mode: Allows to manually move the position of each part on the Nesting board.
  •  Ordering mode: Allows to manually modify the machining order on the Nesting board.
  •  Bridge mode: Allows to manually add bridges between parts.
  •  Numbering: Allows to manually modify the numbering of the parts on the Nesting board.

 

 

Options:

  •  Colors:

          Shows the correlation between the displayed colors and the name of the associated source files.

  •  Options:
    • Options: 
      •  Display parts name: Displays the part names on the Graphics Area of the manual nesting.
      •  Rotation step: Defines the angle at which the part will rotate when the "Part rotation" shortcut is activated.
    • Shortcuts:
      •  Part rotation: (By default "R"). Can be used in the " Moving mode" editing mode to rotate a part by the rotation step. By default "R".
      •  Activate/Desactivate limitation zone: (By default "W"). Can be used in the edit mode " Moving mode" to activate or deactivate the limitation zone, which is an available zone for moving the selected part.
      • Activate/Deactivate assembly colors: (Default is "J"). Shows or hides the colors of the source files.
      •  Change color: (Default is "H"). Allows to change the color assigned to a source file.

 

Sections:

CONTENTS SECTION

Options:

  •  New sheet : Allows to add a sheet (of the material of your choice with the arrow) to the current Nesting project.
  •  Delete : Allows to delete a sheet from the current project. Once the sheet has been deleted, all the parts of this sheet are in the Not nested parts section.
  •  Up/Down : Allows to manage the order of the boards (materials) in the Design Tree.
    •

     

    Design Tree:

     Editable Sheets: List of sheets of the current nesting.

               Board: List of nested sheets sorted by board of the same material.

                        Sheet: Nested sheet of parts to be nested.

                                    Part to be nested: Part of the nested sheet.

     

    EDITION SECTION

    This section is the graphics interface for changing the position of the parts in the selected sheet.

    The editing possibilities are relative to the selected editing mode:

     Moving mode 

    In the mode " Moving mode" the position and the rotation of parts can be edited.

    To move a part, simply click and drag while holding down the click.

    To rotate a part (by the selected rotation step) hold the selected part and press the "R" key.

    A rectangular limitation zone is visible when you press the "W" key, which allows you to position the part as close as possible to the other parts. This zone takes into account the dimensions of the other parts already placed on the sheet, and also the diameter of the milling tool.

    In this editing mode, it is also possible to individually move:

    • The lead-in points of each part, in order to limit the rapid movements of the tool and thus reduce the machining time.

    • The labels of the parts, in order to have a personalized positioning.

     Ordering mode

    In the " Ordering mode" it is possible to manually redefine the machining order of the parts, calculated by the algorithm thanks to the CAM parameters.

    To remove the calculated machining order, simply click on the parts.

    To assign a new order, select the parts in the order of your choice.

     Bridge mode

    In the "Bridge mode", it is possible to manually specify the position of bridges, which allow to keep a connection between two parts and facilitate their holding together during machining.

     It is not possible to insert bridges on curved edges.

    Display:

    By hovering over a part' s edge, it changes color and becomes:

    • Yellow, on a part zone next to another.
    • Orange, on a part zone far enough to other parts.
    • Red, on a part zone where it’s not possible to insert bridges, because it is too close to the ends of the edges or other bridges.

     

    Options:

    In the "Bridge mode" mode, when the  option button is clicked, it is possible to change the bridge settings:

    • Length: Defines the total length of the bridge (at the bottom of the wedges).
    •  Height: Defines the height of the bridge from the bottom of the panel.
    •  Angle: Defines the angle of the bridge from the bottom of the panel.

     

     Numbering

    In the "Numbering" mode it is possible to manually redefine the numbering of the parts, calculated by the algorithm thanks to the Numbering parameters.

    To modify the numbering, you must first activate the " Specific to sheet" option, then activate the None " Mode".

     

    Graphics tools

    There are 3 options in the Graphics Area:

    "Zoom to Fit": Zooms in or out so that the selected sheet or sheets are fully visible.

    "Zoom to Area": Zooms in on a portion of the selected graphics area.

     "Show/hide": Hide or show, in the Graphics Area, the following elements:

           "See order": Shows the machining order of the nesting project (defined from the Machining parameters).

           "See lead-in points": Shows the start and end points of the contour machining of the parts.

           "See bridges": Shows the bridges of the parts, defined automatically in the Bridge rule or manually in the Bridge mode.

          "See Numbering": Shows the numbering of the parts in the nesting sheet (defined from the Numbering parameters).

           "See assembly colors": (Only if Assemblies have been inserted in the source files) Shows the parts from the same assembly with the same color.

           "See part types": Shows the different Part types with the same color.

     

    NOT NESTED PARTS SECTION

    The section of not nested parts can be used as a storage area for parts to be placed later when rearranging the Nesting sheet, in the " Moving mode".
