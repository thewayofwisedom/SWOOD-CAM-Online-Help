PART TYPES

Part types are a set of measures used to improve the way parts are held together on nesting sheets during machining. These measures are as following:

  • Automatic bridges: Computes the position of the bridges on each part to improve their grip.

  • Ordering by size: Defines the machining order of the parts from the smallest to the largest.

  • Optimal lead in/out: Place the lead in/out point of each part in such a way that the parts are held on the sheet as long as possible during machining.

  • Offset: Exclusion zone on the edges of the sheet where small parts cannot be nested.

 

Access the part type configuration window:

In the Nesting projects window, click on the "Parts type" button.

The Part type configuration window consists of 3 main sections (the Part types list, the Global configuration and the Part type Configuration) and a toolbar.

Toolbar:

  •  New (part type): Creates a new part type configuration.
  •   Save: Saves the library.
  •  Delete: Deletes the selected part type configuration.

 

Sections:

  • PART TYPES LIST

In this section there is a list of all the saved " part types" configurations.

It is possible to rearrange their order by a simple drag and drop.

 → With a right click on one of the " part types" configurations, the following actions are available:

                  -  New: Creates a new part type configuration.

                  -  Delete: Deletes the part type configuration.

                  - Copy: Copies the part type configuration with all its parameters.

                  -       Rename: Renames the part type configuration.

 

  • GLOBAL CONFIGURATION

In this section it is possible to modify the following parameters:

Generalities

  •  Name: Name of the part type configuration.
  •  Description : Description of the part type configuration.

Machining

  •  Forced lead in/out on the middle of the longest edge: Forces the position of the tool entry and exit to the middle of the longest edge of the part.
  •  Ordering by size: Defines the machining order of the parts from the smallest to the largest.
  •  Size tolerance: Defines the tolerance (in percentage) in the detection of the size of the parts.

Nesting

  •  Forced four corner: Positions parts differently from normal mode, helping to hold small parts securely on the plate during machining (see Multiple Corner).
  •  Forced four corner on last sheet: (available only if the "Force four corner" parameter is checked) Forces or not the four-corner layout on the last nesting panel, which often has fewer parts and can provide a reusable offcut.

 

Bridge properties

  •  Safety gap: Safety distance between the tool lead in/lead out and any bridge.
  •  Use default properties: Uses the default bridge properties of SWOOD CAM.
  •  Length: (modifiable only if " Use default properties" is unchecked) Defines the total length of the bridge (at the bottom of the corners).
  •  Height: (modifiable only if " Use default properties" is unchecked) Defines the height of the bridge from the bottom of the panel.
  •  Angle: (modifiable only if the parameter " Use default properties" is unchecked) Defines the angle of the bridge from the bottom of the panel.

 

  • PART TYPE CONFIGURATION

In this section there are two zones, on the left a zone for the definition of the different part types and on the right a zone with parameters specific to each part type created.

 

Definition zone for the different part types

In this area, it is possible to create several part types by filling in the following attributes:

  • Name: Name of the part type (e.g. large, medium or small).
  • Maximum surface: Maximum surface area (in mm²) at which the part still belongs to this part type.
  • Maximum width: Maximum width (in mm) at which the part still belongs to this part type.

 

 Tip: It is possible to assign a maximum surface and a maximum width (like the type "Small" in the image above), in this case there will be two criteria for assigning this part type.

It is also possible to assign only one maximum (like the type "Medium" in the image above), in this case there will be only one criterion for assigning this part type.

Finally it is possible to assign no maximum (like the "Large" type in the image above), in which case this part type will be assigned to all parts that do not already have a part type.

 

Part type parameters zone

For each part type created, it is possible to assign different parameters to optimize the holding of the parts according to their type.

Nesting

  •  Offset: Exclusion zone on the edges of the sheet where small parts cannot be nested.

Example of 100mm offset

 

Machining

  •  Optimal lead in/out: Place the lead in/out point of each part in such a way that the parts are held on the sheet as long as possible during machining.

  •  Enable ordering by size: Also available in the Global configuration, which can be defined for the selected part type.

Example of nesting with machining order by size

 

Bridges

  •  Type: Defines the bridge type that will be inserted on the parts:

                          - None: No bridge will be inserted on the parts.

                          - Contour: Bridges will be inserted at regular intervals.

                                            →  Length: Nominal distance between two bridges on the same part.

                                            →  Neighbor shift maximum: Distance from which two bridges merge into one.
