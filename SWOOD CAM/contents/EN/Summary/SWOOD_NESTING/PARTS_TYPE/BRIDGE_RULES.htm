BRIDGE RULES

The Bridge Rules window, accessible from the Part Type configuration window, is used to define the rules for inserting bridges of parts according to each of its edges.

 

To define a Bridge rule you just have to fill in the three rule attributes:

  • Name : Name of the bridge rule to easily distinguish between them.
  • Minimum length : Length at which an edge will start to fit into the bridge rule.
  • Maximum length : Length to which an edge will fit into the bridge rule.

 

To add attachments to a bridge rule, simply:

       1. Select the bridge rule to be edited.

       2. Click on the edge (visible in the 2D View) around the final location of the bridge.

       3. Adjust the exact position of the insertion point using the inverted triangle and the percentage of the edge length.

       4. Adjust the offset (in mm) to the right or left of the bridge from its insertion point.

 

To delete a bridge, simply select it and press the " Del " key on the keyboard.
