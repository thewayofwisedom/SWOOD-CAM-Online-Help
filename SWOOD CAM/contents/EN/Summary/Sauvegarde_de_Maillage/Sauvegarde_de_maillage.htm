Save Mesh

 

Introduction

The Save Mesh window allows you to create a file used to represent a tool in the simulation.

 

Save a mesh

In order to save a mesh, from a SOLIDWORKS part file, go to the menu Tools/ SWOOD CAM / Utility / Save mesh.

After opening the window, a visual of the tool should appear. It is possible to modify the tool in SOLIDWORKS with this window open.

  •  Allows you to save the mesh file: The name of the .efimesh file must have the same name as the library tool linked to it. The file must also be saved in the "Simulation Tools" folder.
  •  Refreshes the display: The tool representation does not update at the same time as the tool is modified in SOLIDWORKS. Use this button to refresh the view.

 

 

To represent a tool

  1. Create a new SOLIDWORKS part.
  2. Save the part file in the « Simulation tools » folder of the SWOOD user.
  3. Create a new sketch with half of the tool profile based on a manufacturing document.
  4. The new sketch’s settings must be equal to the tool’s definition in the SWOOD library.
  5. Put the origin directly on the tool reference (tool corrector).
  6. Revolve the sketch.
