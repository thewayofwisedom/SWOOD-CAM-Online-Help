ACTIVATION AND LICENSE TRANSFERS

 

When using a fixed license, SWOOD CAM requires a serial number as well as a remote activation.

 

  • License transfers (restitution)
  • Activation of a fixed license
  • Allow access to the activation server

 

License transfers

The license transfer allows to return the SWOOD CAM license of the computer. The add-on can no longer be used. This procedure should be used before uninstalling the SWOOD add-on. It can also be used to activate the license on another computer afterwards.

  • To transfer a license, go to the " Tools \ SWOOD CAM \? \ Transfer licenses ".

 

 

Activation of a fixed license

There are 2 possibilities to activate a product, automatically by internet or manually by email.

  • Automatic activation: Enter the serial number and e-mail address, then click on next.
  • Activation by mail:
    It is possible that the sending of an automatic activation request over the Internet is blocked (firewall/server/proxy), it is then necessary to activate the product by email.
    • Choose "Manually by email" and click "Next" to save the activation request.
    • Send the saved file by email to activation@eficad.com.
    • Wait for the response by return mail of the response file Response.txt and load the file using the "Open" button.
    • Then repeat the operation a second time if it is the first activation of the station.

 

 

Allow access to the activation server (IT department)

In case the computer configuration prohibits access to the activation server ("Cannot connect to activation server" message), two solutions can be considered.

  • Solution 1 : Create an exception in the firewall for the IP address 82.127.168.94 corresponding to the domain activation@eficad.com.
  • Solution 2 : Add the hostname in the DNS server (Example with the following link : http://helpdeskgeek.com/windows-7/windows-7-hosts-file/)
