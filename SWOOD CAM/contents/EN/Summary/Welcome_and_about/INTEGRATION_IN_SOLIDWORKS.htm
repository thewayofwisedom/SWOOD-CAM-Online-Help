INTEGRATION IN SOLIDWORKS

 

SWOOD CAM is a complement fully incorporated in SOLIDWORKS. It is used as an add-on to the software in the same way as the SOLIDWORKS add-ons..

We find the integration of SWOOD in SOLIDWORKS also at the level of accessibility of its features in different menus and libraries identical to that of SOLIDWORKS.

 

This part is composed of several sections:

  • Activating and deactivating the complement
  • First use and choice of data directory
  • Changing the data directory

 

 

Activating and deactivating the complement

As SWOOD CAM is integrated in SOLIDWORKS, its activation and deactivation are therefore possible in the same way via the Options/Add-ins menu..

 Activation is identical to that of the other supplements, namely :

  • If checked on the left, activation in the current SOLIDWORKS session.
  • If checked on the right, permanent activation of the complement.

First use and choice of data directory

At first startup, SWOOD requires the user to set some options in the module.

This data directory can be created locally on the hard disk. (Prefer a drive other than the main C drive), on a server(for a link between several users), or in the SOLIDWORKS EPDM.

These options are defined by the following parameters:

  1. Général
    1. Data directory : Contains all design and machining data. The selection is made by browsing and selecting the folder containing the library of your SWOOD products.
    2. Working directory :  Default path to save your designs.
    3. Language : Defines the language of the complement.
  2. Units
    1. Use metric system : Enables or disables the metric system.
    2. Hide units symbol : Allows to disable the display of symbols (m,mm,..) in SWOOD windows.
  3. Design
    1. Currency : Choice of the currency to be used.

Changing the data directory

If you change the location of the data directory, change the path in the SWOOD CAM options through the following menu: "Tools / SWOOD CAM / Options".
