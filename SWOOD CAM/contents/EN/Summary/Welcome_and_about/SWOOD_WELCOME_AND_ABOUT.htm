WELCOME IN THE HELP OF SWOOD CAM

 

SWOOD CAM is a SOLIDWORKS Add-In dedicated to woodworking that help you to create CNC programs and manufacturing documents.

 

SWOOD CAM is composed of custom libraries of customizable components :

  • Tools

    Create your tools library corresponding to tools in your CNC (name, reference, number, corrector, feed rate ...).

  • Drilling block and aggregates

    Create your aggregates library with possibility to manage multiple spindle, multiple heads, axis rotation ...

  • Machinings

    Create machinings library for usual machining and easy application on parts.

  • Simulation with material removal

    Use the material removal simulation mode with collision detection.

 

 

Create programs directly in SOLIDWORKS parts or assembly and encode global project with manufacturing documents generation.
