MATERIAL REMOVAL SIMULATION

 

Introduction

Material Removal Simulation is a simulation allowing several modes:

  • Material removal during simulation
  • Comparison of the simulation result with the SOLIDWORKS model
  • Collision detection

To access this simulation mode, right-click on the  Program in the SWOOD CAM programming tree and select Simulate RM  .

 

Interface

The interface below will be displayed on the screen to configure the simulation mode as well as to activate it.

This is composed of the following elements:

  •  Show/Hide: Allows you to show or hide certain items during simulation.
    • Tool: Show/Hide the tool during simulation
    • Trajectory: Show/Hide trajectories during simulation
    • Model: Show/Hide the SOLIDWORKS model.
    • Simulation: Show/Hide the result of the simulation, only the SOLIDWORKS model will be visible.
    •  Scrap: Show/Hide the scraps (after using the scrap detection mode).
    •   Origines: Show/Hide programme origins.
  • Compare result with your model: Allows you to compare the result of the simulation with the SOLIDWORKS model.
  • Detec scrap: Allows you to calculate scraps to show/hide them.
  • Detect collisions: Executes the collision detection algorithm.
  •  Settings: Allows you to configure the display settings, refer to the Settings.

 

Simulation

You can simulate the program, a tool or a machining operation as you wish. It is also possible not to simulate but to display only the result of the simulation.

To activate simulation, right-click on the program, tool, or machining operation to be simulated in the interface and select Simulate . To view only the result, select Show Result.

Detection of collisions

When the  Detect collisions has been enabled, SWOOD performs an analysis to check whether collisions have been detected. Any non-sharp part of the tool colliding with the part, or an element of the table (suction cups or clamps) will be listed.

When collisions are detected, only the operations involved are displayed, along with the items involved.
