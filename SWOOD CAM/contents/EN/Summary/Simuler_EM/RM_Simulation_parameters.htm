SETTINGS

 

In Simulation mode with material removal, several parameters can be changed. Some of them require lengthy calculations, so it is preferable to use them only for specific cases, and with a good machine configuration.

  • Precision: This parameter modifies the size of the voxels, corresponding to the decomposition of the mesh. The smaller the voxel, the longer the calculation time.
    • High: Best precision
    • Medium: Combine performance and precision
    • Low: Reduce precision to imporve performance
  • Manual settings: If checked, the previous setting "Precision" is disabled.This setting allows you to manually change the size of the voxels.
  • Tolerance level: Allows you to change the maximum distance between the model and the tool for the voxel to be taken into account.
  • Enable/Disable: Enables or defaults the scraps detection settings and collision display.
  • Color: Change the display of the simulation rendering. It is possible to change the colour of the outer part of the part, the body of the part, as well as the scraps.
