GETTING STARTED - RESIZING

With SWOOD CAM, if you modify the part you programmed, the SWOOD CAM program will update itself. 

 

28. For example, update the length of the panel by changing the dimensioning sketch of the panel. 

To do so, select the panel and change the length from 900 to 1000 mm. 

If you have SWOOD Design, you can use the feature  Edit Panel in the SWOOD Design toolbar and change the length value. 

 

29. On the  SWOOD CAM tab, click on  Rebuild to recalculate the SWOOD CAM tool paths. 

 

30.  Simulate the program to check the update.
