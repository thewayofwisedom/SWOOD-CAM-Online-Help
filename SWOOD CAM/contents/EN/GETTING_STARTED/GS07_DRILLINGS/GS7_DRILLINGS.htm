GETTING STARTED - DRILLINGS

We follow with the drilling the panel.

 

18. To do so be sure you still are in the OP0 origin in the SWOOD CAM tree and select " Drilling" in the SWOOD CAM tool bar. 

 

19. Select the aggregate " DB-DRILLING_BLOCK" you want to use for drilling select "Drill" in the operation list then "Add".

 

 

20. Here below are the drill parameters. According to the composition of the selected drilling block, SWOOD automatically detects which spindle to use to drill all the possible holes. In this case, SWOOD selects the Ø5 diameter spindle. 

 You can find help for the drill operation by clicking here.

 SWOOD selects only one spindle instead of multiple Ø5 spindle. That is a post processor option.

 

21. Without changing anything, validate the Drilling machining .

 Note: The Drilling machining (that contains the Drill operation) is added the current milling phase.
