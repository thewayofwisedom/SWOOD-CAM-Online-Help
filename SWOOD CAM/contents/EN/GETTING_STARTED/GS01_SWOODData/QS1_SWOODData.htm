GETTING STARTED - SWOODData

In SWOOD, the SWOODData is a database with all the libraries concerning Design and Cam. It is very important to fill it up correctly, if not, no access will be available like frames, tools, connectors etc. A standard SWOODData is provided by your reseller.

 

 

1. To indicate the path of the SWOODData in SOLIDWORKS you have to go to: Tools→ SWOOD CAM →  Settings  

2. A new window opens, on the section ‘’Data Directory’’. It is necessary to replace the SWOODData folder to have access to the database:

 

3. There is another tab to be filled is the ''Working directory'' : this is the folder where projects will be stored. In the same way as the data directory, it must be filled in:

After filling in these 2 folders you can start using SWOOD.
