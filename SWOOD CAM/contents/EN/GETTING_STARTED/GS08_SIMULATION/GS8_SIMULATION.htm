GETTING STARTED - SIMULATION

At this stage, we have programmed all the part with calibrating and drilling it. 

To check if everyhing works, it's possible to simulate the program created. SWOOD has two ways to simulate it.

 

22. The first is by clicking on the  Simulate in the SWOOD CAM tab. 

 

23. The second one is by  right clicking the  milling phase and select  Simulate RM (Remove material).

 You can find help for the Remove Material Simulation by clicking here.

 

24. Right click on the  Milling phase level. You can either simulate the whole tool path or just the result. Select "Result display".

 

25. Then to check if everything is correct, click on " Compare with SOLIDWORKS model". 

 

26. This verification tool will compare the SOLIDWORKS model and check if the tool path does not machine wrong parts of the panel. Here everything is correct.

Blue it means that there is missing material to remove, red there is too much material removed.

 

27. Validate the simulation .
