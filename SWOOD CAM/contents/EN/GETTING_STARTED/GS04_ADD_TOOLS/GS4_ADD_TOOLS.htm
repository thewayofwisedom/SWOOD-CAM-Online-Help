GETTING STARTED - ADD TOOLS

We now need to add the tools that will be used to contour the part. 

 
You can find help to manage the tools by clicking here.

 

9.  To do so, open the SWOOD CAM pane on the SOLIDWORKS Taskpane tab and select the tools library.

 You can find help for the SWOOD CAM Library by clicking here 

 

10. To add a tool the current milling phase, the quickest way is to drag and drop the tool from the tool library to the SOLIDWORKS display section. 

So in the " GCODE" section open the  Generic Cutter folder and then select the tool " Roughing Ø18".

 

11. Drag and drop it from the SWOOD CAM pane to the SOLIDWORKS display section.

 Do not drop the tool ON the part. Drop it around the 3D model. 

 

 Note: The SWOOD CAM tree now contains the tool ‘’Roughing Ø18’’
