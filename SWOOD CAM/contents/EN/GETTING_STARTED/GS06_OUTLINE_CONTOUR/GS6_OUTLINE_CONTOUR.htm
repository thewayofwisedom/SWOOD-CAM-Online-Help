GETTING STARTED - OUTLINE CONTOUR

Now that we have created a  Milling phase,  positioned the part and then added  tools and  aggregates, we can now create the machining. 

We start by calibrating the panel.

 

13. To do so, be sure you are in the right origin. Therefore click on  OP0 in the SWOOD CAM tree. 

The direction of the Z-axis (in blue) must be opposite to the tool diving direction.

This determines the face that will be machined.

 

14. Then click on  Outline in the SWOOD CAM toolbar. 

 

15. In the opened window, click on the tool " T6 - Roughing Ø18" then select "Contour" and then "Add". 

To resume, you have selected which entity you want to mill (the  Outline).
You have selected the tool to use (the  roughing cutter). 
Then SWOOD has showed you all the available operations. 

You chose which one you want to add to the current milling phase.

 

16. You can see all the contour operation parameters. Change the bottom exceeding distance and set it to 0 mm. 

 You can find help for the contour operation by clicking here.

 Note: The Outline machining (that contains the contour operation) is added the current milling phase.
