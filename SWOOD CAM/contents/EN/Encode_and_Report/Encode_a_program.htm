ENCODE

 

Programs can be created for several machine languages. The choice of language depends on the post-processor used.

To encode a program click on the  Encodage button on the toolbar of the SWOOD CAM tab, or right-click on the name of the phase in the SWOOD CAM tree and select the  Encode command.

 

SWOOD CAM generates a program in a new window, depending on the post-processor selected in the Phase Management.

When coding, the program is automatically saved in the folder where the SOLIDWORKS part is located and with the default part name.

From the coding window, it is possible to:

  • Save : Saves the program if a manual change is made in the window.
  • Save as : Saves a copy of the programme in a folder of your choice.
  • Print : Print the programme.
  • Open : If the program file extension is linked to a machine software (such as WOODWOP, BIESSEWORKS, HOPS ...), open it in the machine software interface.
  • Open the folder : Opens the folder in Windows Explorer.
  • Transfert : Allows a COM port transfer module to be used.
