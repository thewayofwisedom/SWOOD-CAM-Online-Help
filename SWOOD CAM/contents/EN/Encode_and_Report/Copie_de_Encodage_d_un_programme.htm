REPORT

 

Introduction

The Report window is the report generation window. It appears after selecting the feature  SWOOD CAM Reporting.

It allows you to generate a report of the current project in different formats.

It is composed of three main parts: 

  1. Tree structure
  2. Visualization
  3. Data

 

 

Tree structure

The tree structure is divided into different levels: 

  1.  First level
    1.  All documents generated and accessible
    2. / List of project parts/assemblies
      1.  List of machining programs
        1.  Machining programs
      2.  Panel definition
        1.  List of panel edgebands
          1.  Definition of edgebands
        2.  List of panel stock
          1.  stock Dimensions

 

 

Visualization

By default this part is automatically displayed as soon as the report is generated. It is composed of two sections: 

  1. The first one is a 3D view section where the whole project is displayed. When a part or an assembly is selected in the tree view, it is highlighted in red in order to visualize it in the project. 
  2. The second section is an image providing more information [Example: Selecting a stock → Image of the stock with dimensions]. This generated image is used in the report documents. By right-clicking on this image, the image settings will appear.

 

 

Data

This part displays the status and the name of all the parameters of the level selected in the tree structure. These parameters are compiled to generate the report documents.
